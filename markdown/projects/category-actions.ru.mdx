## Действия с категориями

Работа с категориями проектов включает следующие действия:

- Создание категории проекта — <span className="font-normal">добавление новой категории проектов</span>
- Редактирование категории проекта — <span className="font-normal">изменение названия существующей категории</span>
- Удаление категории проекта — <span className="font-normal">удаление категории из системы</span>

---

### Добавление категории

Чтобы добавить категорию, перейдите на страницу <b>«Категории проектов»</b>, нажмите кнопку <b>«Создать новую»</b> в верхней части страницы, в появившемся модальном окне введите название категории проекта.

### Управление категорией

Чтобы выполнить действия с существующей категорией, нажмите кнопку <b>«Меню действий»</b> (иконка с тремя точками) справа в элементе списка.

#### Редактирование категории

Чтобы отредактировать категорию в <b>«Меню действий»</b> выберите пункт <b>«Редактировать»</b>. После этого откроется окно редактирования, в котором можно изменить название категории. Название категории не может быть пустыми.

#### Удаление категории

Чтобы удалить категорию в <b>«Меню действий»</b> выберите пункт <b>«Удалить»</b>. В появившемся модальном окне нажмите кнопку <b>«Удалить»</b>.
