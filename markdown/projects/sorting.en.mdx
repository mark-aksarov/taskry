## Project sorting

To sort projects, click the <b>“Sorting”</b> button at the top of the page. The following sorting options are available in the menu:

- By creation date — <span className="font-normal">sorts from newest to oldest.</span>
- By name — <span className="font-normal">sorts in alphabetical order.</span>
- By deadline — <span className="font-normal">sorts by the nearest deadline.</span>
- By status — <span className="font-normal">sorts by project state: “Active”, “Pending”, “Completed”.</span>
