## Фильтры проектов

Чтобы отфильтровать проекты, нажмите кнопку <b>«Фильтры»</b> в верхней части страницы. Откроется модальное окно фильтрации, где доступны следующие фильтры:

- Нет активных задач — <span className="font-normal">отображает только проекты без активных задач.</span>
- Дедлайн с / по — <span className="font-normal">позволяет выбрать диапазон дат.</span>
- Статус — <span className="font-normal">«В ожидании», «Активный», «Завершён».</span>
- Категория — <span className="font-normal">например: «Дизайн», «Внутренние системы», «Веб‑разработка».</span>
- Клиент — <span className="font-normal">список всех клиентов, связанных с проектами.</span>
- Создатель — <span className="font-normal">список пользователей, создавших проекты</span>
