## Project Creation

To create a project, go to the “Projects” page, click the <b>“Create new”</b> button at the top of the page, and in the dropdown menu select <b>“Project”</b>.

Creating a project includes filling in the following data:

- Title — <span className="font-normal">a short project name</span>
- Deadline — <span className="font-normal">the project completion due date</span>
- Status — <span className="font-normal">the current state of the project</span>
- Description — <span className="font-normal">additional information about the project</span>
- Client — <span className="font-normal">who the customer is or for whom the project is being delivered</span>
- Category — <span className="font-normal">a group for better project organization</span>

After saving, the project will appear in the project list if it matches the current sorting, filters, and search conditions.
