## Project view

The system supports two project display modes. In <b>“List”</b> mode, data is displayed as list items. In <b>“Cards”</b> mode, projects are presented as a grid, where each project is shown as a separate card. Both modes use the same dataset but differ in how the information is visualized.

---

### List

To switch to list mode, click the <b>“List”</b> button in the view switcher at the top of the page.

### Actions

Both project cards and list items support the same set of actions:

- Clicking the project title <span className="font-normal">opens a side panel with project details: description, deadline, status, category, client, and project author. This allows quick access to full information without navigating to a separate page.</span>
- Clicking the name or avatar of the project creator <span className="font-normal">opens a user side panel: role, contacts, short description, and other details. This helps quickly understand the user’s role and navigate to their profile if needed.</span>
- Clicking the “Comments” button <span className="font-normal">opens a modal with project comments.</span>
- Clicking the “Actions menu” <span className="font-normal">opens a list of available actions.</span>

Additionally, list items support an extra action:

- Clicking the client name or avatar <span className="font-normal">opens a side panel with client details: contact information, company, short description, and other data. This allows quick access to client information and navigation to their profile if needed.</span>
