## Project Filters

To filter projects, go to the <b>“Projects”</b> page, click the <b>“Filters”</b> button at the top of the page. A filtering modal window will open, where the following filters are available:

- No active tasks — <span className="font-normal">displays only projects without active tasks.</span>
- Deadline from / to — <span className="font-normal">allows you to select a date range.</span>
- Status — <span className="font-normal">“Pending”, “Active”, “Completed”.</span>
- Category — <span className="font-normal">for example: “Design”, “Internal Systems”, “Web Development”.</span>
- Client — <span className="font-normal">a list of all clients associated with projects.</span>
- Creator — <span className="font-normal">a list of users who created the projects.</span>
