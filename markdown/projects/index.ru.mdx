## Проекты

Проект — это сущность, которая хранит основную информацию о работе и объединяет связанные с ней данные:

- Основная информация — <span className="font-normal">название, описание, дедлайн</span>
- Задачи — <span className="font-normal">единицы работы внутри проекта</span>
- Комментарии — <span className="font-normal">обсуждения, привязанные к проекту</span>
- Клиент — <span className="font-normal">информация о заказчике или организации</span>
- Категория — <span className="font-normal">группировка проекта по типу или направлению</span>
- Статус — <span className="font-normal">текущее состояние проекта</span>
- Автор — <span className="font-normal">пользователь, создавший проект</span>

---

### Просмотр и навигация

Проекты отображаются в виде списка или сетки на странице <b>«Проекты»</b>. Чтобы перейти в этот раздел, необходимо выбрать пункт <b>«Проекты»</b> в боковой навигации приложения.

Также просмотр и редактирование проекта доступны на странице детализации проекта. Для этого необходимо нажать на заголовок проекта и в боковой панели нажать кнопку <b>«Открыть проект»</b>. На этой странице можно изменять каждое поле в отдельном модальном окне.

---

### Действия

С проектами доступны следующие основные действия:

- Создание проекта — <span className="font-normal">добавление нового проекта с заполнением базовой информации: название, дедлайн, клиент, категория и другие параметры</span>
- Редактирование проекта — <span className="font-normal">изменить данные существующего проекта</span>
- Удаление проекта — <span className="font-normal">удаление одного или нескольких проектов из системы</span>
- Изменение статуса — <span className="font-normal">обновление текущего состояния проекта</span>
