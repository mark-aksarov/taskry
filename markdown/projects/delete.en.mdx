## Deleting a project

The system supports two types of deletion: <b>single</b> and <b>bulk</b>. Both options allow you to permanently remove projects from the system.  
Project deletion is irreversible, so before deleting, make sure the project is no longer needed and all important data has been saved.

---

### Single deletion

To delete a project, go to the <b>“Projects”</b> page, click the <b>“Actions menu”</b> button (three-dot icon) on the right side of the card or list item, and select <b>“Delete”</b> from the menu that appears.

### Bulk deletion

To delete multiple projects at once, select the required projects using the checkboxes. Then click the <b>“Actions”</b> button at the top of the page and choose <b>“Delete”</b>.
