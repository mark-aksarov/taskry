## Users

Managing users and their access rights is a key part of administration in Taskry.

---

### User Roles

Taskry supports multiple access levels for different types of users.

---

### Available Roles

Assign appropriate roles to users:

#### 1. Administrator

Full project access, can manage members and settings.

#### 2. Editor

Can create, edit, and delete tasks.

#### 3. Viewer

Can only view project and task information.

---

### Managing Access

Change user roles through the "Manage Members" section. Each user receives a notification about their role change.
