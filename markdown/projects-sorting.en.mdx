## Project sorting

To sort projects, click the <b>“Sorting”</b> button in the top panel of the list. In the menu that opens, the following sorting options are available:

- By creation date — <span className="font-normal">from newest to oldest or vice versa.</span>
- By name — <span className="font-normal">in alphabetical order.</span>
- By deadline — <span className="font-normal">by nearest or latest due date.</span>
- By status — <span className="font-normal">grouping by project state: “Active”, “Pending”, “Completed”.</span>

![image](https://storage.yandexcloud.net/task-manager-docs/project_sorting_en.jpg)

After selecting a parameter, the project list will update and be displayed according to the chosen sorting method.
