## Задачи

Задачи — это основные рабочие единицы в Taskry, которые помогают отслеживать прогресс работы.

---

### Что такое задача?

Задача представляет собой отдельный элемент работы, который нужно выполнить. Каждая задача может иметь приоритет, сроки, исполнителя и множество других атрибутов.

---

### Создание задачи

Создавайте задачи для отслеживания работы:

#### 1. Откройте проект

Выберите нужный проект из списка.

#### 2. Нажмите "Добавить задачу"

Введите название и описание задачи.

#### 3. Установите параметры

Назначьте исполнителя, установите приоритет и срок выполнения.

---

### Статусы задач

Задачи могут находиться в различных статусах: новая, в работе, на проверке, завершена. Используйте статусы для отслеживания прогресса.
