## Project deletion

The system provides two types of deletion: <b>single</b> and <b>bulk</b>. Both options allow you to permanently remove projects from the system.  
Project deletion is an irreversible action, so before deleting, it is recommended to make sure the project is no longer needed and that all important data has been saved.

---

### Single deletion

To delete a project, click the <b>“Action menu”</b> button (three dots icon) on the right side of the project card or list item, and select <b>“Delete”</b> from the dropdown menu. In the modal window that appears, click the <b>“Delete”</b> button.

![image](https://storage.yandexcloud.net/task-manager-docs/project_single_deletion_en.jpg)

After confirmation, the project will be permanently deleted and removed from the list.

---

### Bulk deletion

To delete multiple projects at once, select the required projects using checkboxes. Then click the <b>“Actions”</b> button and choose <b>“Delete”</b>. In the modal window that appears, click the <b>“Delete”</b> button.

![image](https://storage.yandexcloud.net/task-manager-docs/project_multiple_deletion_en.jpg)

After confirmation, all selected projects will be deleted and removed from the list.
