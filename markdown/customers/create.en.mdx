## Creating a client

To create a client, go to the <b>“Clients”</b> page, click the <b>“Create new”</b> button at the top of the page, and select <b>“Client”</b> from the dropdown menu.

Creating a client includes filling in the following data:

- Full name — <span className="font-normal">the display name of the client.</span>
- About — <span className="font-normal">short information about the client shown in the profile.</span>
- Email — <span className="font-normal">email address for contacting the client.</span>
- Phone number — <span className="font-normal">client’s contact phone number.</span>
- Public link — <span className="font-normal">link to the client’s profile.</span>
- Address — <span className="font-normal">client’s location or address.</span>
- Company — <span className="font-normal">the company associated with the client.</span>

After saving, the client will appear in the clients list if it matches the current sorting, filters, and search conditions.
