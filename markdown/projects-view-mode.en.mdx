## Project display

The system supports two project display modes. In <b>"List"</b> mode, data is displayed in structured rows and columns. In <b>"Cards"</b> mode, projects are presented as a grid of cards, where each item is shown as a separate block. Both modes use the same dataset but differ in the way they are visualized.

---

### List

To switch to list mode, click the <b>"List"</b> button in the view switcher at the top of the page.

![image](https://storage.yandexcloud.net/task-manager-docs/project_list_en.jpg)

Each item in the list contains the following data

- Project title — <span className="font-normal">short title</span>
- Deadline — <span className="font-normal">completion date</span>
- Created by — <span className="font-normal">who created the project</span>
- Client — <span className="font-normal">linked customer</span>
- Category — <span className="font-normal">project type</span>
- Company — <span className="font-normal">client organization</span>
- Status — <span className="font-normal">current state</span>

---

### Cards

To switch to list mode, click the <b>"Cards"</b> button in the view switcher at the top of the page.

![image](https://storage.yandexcloud.net/task-manager-docs/project_grid_en.jpg)

Each project card contains the following data

- Project title — <span className="font-normal">short title</span>
- Deadline — <span className="font-normal">completion date</span>
- Status — <span className="font-normal">current state</span>

---

### Interactive actions

- Clicking on the project title
  <div>
    Opens a sidebar with project details: description, deadline, status,
    category, client, and creator. Allows quick access to full information
    without navigating to a separate page.
  </div>
- Clicking on the project creator's name
  <div>
    Opens a user sidebar: position, contacts, short description, and other
    details. Helps quickly understand the employee's role and, if necessary,
    navigate to their profile.
  </div>
- Clicking on the client name
  <div>
    Opens a client sidebar: contact details, company, short description, and key
    information. Allows getting context about the client and navigating to their
    card.
  </div>
- "Comments" button
  <div>
    When clicking <b>"Comments"</b>, a modal window with project comments opens.
  </div>
- Actions menu
  <div>
    When clicking the <b>"Actions menu"</b> button, a list of available actions
    opens
  </div>
