## Project creation

To create a project, click the <b>"Create new"</b> button and select <b>"Project"</b> from the menu that appears.

![image](https://storage.yandexcloud.net/task-manager-docs/project_creation_en.jpg)

Creating a project includes filling in the following data:

- Project title (required)
- Deadline (required)
- Status (required)
- Description (optional)
- Customer (optional)
- Category (optional)

After saving, the project will appear in the project list if it matches the current sorting, filters, and search conditions.
