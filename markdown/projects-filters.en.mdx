## Project filters

To filter projects, click the <b>“Filters”</b> button in the top panel. A filtering panel will open, where the following parameters are available:

- No active tasks — <span className="font-normal">shows only projects without active tasks.</span>
- Deadline from / to — <span className="font-normal">allows you to select a date range.</span>
- Status — <span className="font-normal">“Pending”, “Active”, “Completed”.</span>
- Category — <span className="font-normal">for example: “Design”, “Internal Systems”, “Web Development”.</span>
- Customer — <span className="font-normal">a list of all customers associated with projects.</span>
- Creator — <span className="font-normal">a list of users who created the projects.</span>

![image](https://storage.yandexcloud.net/task-manager-docs/project_filters_en.jpg)

After selecting the required parameters, click <b>“Apply filters”</b>, and the project list will update according to the selected conditions.
