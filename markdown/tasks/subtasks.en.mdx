## Subtasks

Subtasks allow you to break a main task into smaller steps for easier management and progress tracking.

Each subtask contains the following data:

- Text — <span className="font-normal">a short description of the subtask.</span>
- Status — <span className="font-normal">indicates whether the subtask is done or undone.</span>

---
