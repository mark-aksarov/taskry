## Category actions

Working with task categories includes the following actions:

- Create task category — <span className="font-normal">add a new task category</span>
- Edit task category — <span className="font-normal">change the name of an existing category</span>
- Delete task category — <span className="font-normal">remove a category from the system</span>

---

### Creating a category

To create a category, go to the <b>“Task categories”</b> page, click the <b>“Create new”</b> button at the top of the page, and in the modal window enter the name of the task category.

#### Editing a category

To edit a category, in the <b>“Actions menu”</b> select <b>“Edit”</b>. After that, an edit window will open where you can change the category name. The category name cannot be empty.

To delete a category, in the <b>“Actions menu”</b> select <b>“Delete”</b>. In the modal window that appears, click the <b>“Delete”</b> button.
