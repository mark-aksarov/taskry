## Отображение проектов

Система поддерживает два режима отображения проектов. В режиме <b>«Список»</b> данные выводятся с упорядоченными строками и колонками. В режиме <b>«Карточки»</b> проекты представлены в виде сетки карточек, где каждый элемент отображается как отдельный блок. Оба режима используют один и тот же набор данных, но различаются способом визуализации.

---

### Список

Чтобы переключиться в режим списка, нажмите кнопку <b>«Список»</b> в панели переключения вида в верхней части страницы.

Каждый элемент в списке содержит следующие данные:

- Заголовок — <span className="font-normal">краткое название проекта</span>

- Статус — <span className="font-normal">текущее состояние проекта</span>
- Кем создан — <span className="font-normal">кто создал проект</span>
- Клиент — <span className="font-normal">информация о заказчике</span>
- Категория — <span className="font-normal">группа для организации проектов</span>
- Компания — <span className="font-normal">организация клиента</span>

---

### Карточки

Чтобы переключиться в режим списка, нажмите кнопку <b>«Карточки»</b> в панели переключения вида в верхней части страницы.

### Интерактивные действия

- Нажатие на заголовок проекта <span className="font-normal">открывает боковую панель с деталями проекта: описание, дедлайн, статус,
  категория, клиент и создатель. Позволяет быстро просмотреть полную
  информацию без перехода на отдельную страницу.</span>
- Нажатие на имя создателя проекта <span className="font-normal">открывает боковую панель пользователя: должность, контакты, краткое описание
  и другие сведения. Помогает быстро понять роль сотрудника и при
  необходимости перейти к его профилю.</span>
- Нажатие на имя клиента <span className="font-normal">открывает боковую панель клиента: контактные данные, компания, краткое
  описание и ключевая информация. Позволяет получить контекст о клиенте и
  перейти к его карточке.</span>
- Нажатие на кнопку «Комментарии» <span className="font-normal">открывает модальное окно с комментариями проекта.</span>
- Нажатие на «Меню действий» <span className="font-normal">открывает список доступных действий.</span>
