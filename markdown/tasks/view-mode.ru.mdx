## Просмотр задачи

Система поддерживает два режима отображения задач. В режиме <b>«Список»</b> данные выводятся в виде элементов списка. В режиме <b>«Карточки»</b> задачи представлены в виде сетки, где каждая задача отображается как отдельная карточка. Оба режима используют один и тот же набор данных, но различаются способом визуализации.

---

### Список

Чтобы переключиться в режим списка, перейдите на страницу <b>«Задачи»</b>, нажмите кнопку <b>«Список»</b> в панели переключения вида в верхней части страницы.

После переключения в основной области страницы будет отображен список задач, каждая задача содержит следующие данные:

- Заголовок — <span className="font-normal">краткое название задачи</span>
- Дедлайн — <span className="font-normal">срок завершения задачи</span>
- Статус — <span className="font-normal">текущее состояние задачи</span>
- Исполнитель — <span className="font-normal">изображение или полное имя пользователя, ответственного за выполнение задачи</span>
- Категория — <span className="font-normal">группа для организации задачи</span>
- Проект — <span className="font-normal">проект, к которому относится задача</span>

### Карточки

Чтобы переключиться в режим карточек, нажмите кнопку <b>«Карточки»</b> в панели переключения вида в верхней части страницы.

### Интерактивные действия

- Нажатие на заголовок задачи <span className="font-normal">открывает боковую панель с деталями задачи: описание, дедлайн, статус, категория, исполнитель, проект, список подзадач. Позволяет быстро просмотреть полную информацию без перехода на отдельную страницу.</span>
- Нажатие на имя исполнителя задачи <span className="font-normal">открывает боковую панель пользователя: должность, контакты, краткое описание и другие сведения. Помогает быстро понять роль сотрудника и при необходимости перейти к его профилю.</span>
- Нажатие на заголовок проекта <span className="font-normal">открывает боковую панель с деталями проекта: описание, дедлайн, статус, категория, клиент и автор проекта. Позволяет быстро просмотреть полную информацию без перехода на отдельную страницу.</span>
- Нажатие на кнопку «Комментарии» <span className="font-normal">открывает модальное окно с комментариями задачи.</span>
- Нажатие на «Меню действий» <span className="font-normal">открывает список доступных действий.</span>
