## Task deletion

The system supports two types of deletion: <b>single</b> and <b>bulk</b>. Both options permanently remove tasks from the system.
Task deletion is an irreversible action, so it is recommended to ensure that the task is no longer needed and that all important data has been saved before proceeding.

---

### Single deletion

To delete a task, go to the <b>"Tasks"</b> page, click the <b>"Actions Menu"</b> button (three dots icon) on the right side of a card or list item, and select <b>"Delete"</b> from the dropdown menu.

### Bulk deletion

To delete multiple tasks at once, select the required tasks using checkboxes. Then click the <b>"Actions"</b> button at the top of the page and choose <b>"Delete"</b> from the menu.
