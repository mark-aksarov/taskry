## Tasks

A task is a unit of work that contains the data required to complete a specific action within a workflow:

- Title — <span className="font-normal">a short name of the task</span>
- Description — <span className="font-normal">detailed information about the task</span>
- Deadline — <span className="font-normal">the date by which the task must be completed</span>
- Project — <span className="font-normal">the project the task belongs to</span>
- Assignee — <span className="font-normal">the user responsible for completing the task</span>
- Author — <span className="font-normal">the user who created the task</span>
- Category — <span className="font-normal">a logical grouping of the task</span>
- Status — <span className="font-normal">the current state of the task</span>
- Subtasks — <span className="font-normal">breaking the task into smaller parts</span>
- Comments — <span className="font-normal">discussions and notes related to the task</span>

---

### Viewing and navigation

Tasks are displayed as a list or grid on the <b>“Tasks”</b> page. To navigate to this section, select <b>“Tasks”</b> in the application's sidebar.

You can also view and edit a task on the task details page. To do this, click on the task title and then click the <b>“Open Task”</b> button in the side panel. On this page, each field can be edited in a separate modal window.

---

### Actions

The following main actions are available for tasks:

- Create task — <span className="font-normal">add a new task by filling in basic information such as title, deadline, assignee, status, category, and other parameters</span>
- Edit task — <span className="font-normal">update the data of an existing task</span>
- Delete task — <span className="font-normal">remove one or multiple tasks from the system</span>
- Change status — <span className="font-normal">update the current state of a task</span>

Working with subtasks includes the following actions:

- Add subtask — <span className="font-normal">create a new subtask within a task</span>
- Edit subtask — <span className="font-normal">update the data of an existing subtask</span>
- Toggle subtask status — <span className="font-normal">change the state of a subtask (done / undone)</span>
- Delete subtask — <span className="font-normal">remove a subtask from a task</span>

Working with task categories includes the following actions:

- Create task category — <span className="font-normal">add a new task category</span>
- Edit task category — <span className="font-normal">change the name of an existing category</span>
- Delete task category — <span className="font-normal">delete a category from the system</span>
