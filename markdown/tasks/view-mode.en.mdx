## Project display

The system supports two project display modes. In <b>"List"</b> mode, data is displayed in structured rows and columns. In <b>"Cards"</b> mode, projects are presented as a grid of cards, where each item is shown as a separate block. Both modes use the same dataset but differ in the way they are visualized.

---

### List

To switch to list mode, click the <b>"List"</b> button in the view switcher at the top of the page.

- Category — <span className="font-normal">group for organizing projects</span>
- Company — <span className="font-normal">customer's organization</span>

### Cards

To switch to list mode, click the <b>"Cards"</b> button in the view switcher at the top of the page.

### Interactive actions

- Clicking on the project title <span className="font-normal">opens a sidebar with project details: description, deadline, status, category, customer, and creator. It allows you to quickly view full information without navigating to a separate page.</span>
- Clicking on the project creator’s name <span className="font-normal">opens a user sidebar: position, contacts, short description, and other details. It helps quickly understand the employee’s role and navigate to their profile if needed.</span>
- Clicking on the customer’s name <span className="font-normal">opens a customer sidebar: contact details, company, short description, and key information. It provides context about the customer and allows you to navigate to their profile.</span>
- Clicking the “Comments” button <span className="font-normal">opens a modal window with project comments.</span>
- Clicking “Actions menu” <span className="font-normal">opens a list of available actions.</span>
