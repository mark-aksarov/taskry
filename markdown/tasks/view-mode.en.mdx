## Task view

The system supports two task display modes. In <b>"List"</b> mode, data is displayed as a list of items. In <b>"Cards"</b> mode, tasks are presented as a grid, where each task appears as a separate card. Both modes use the same dataset but differ in visualization.

---

### List

To switch to list mode, go to the “Tasks” page, click the <b>"List"</b> button in the view switcher at the top of the page.

- Deadline — <span className="font-normal">task due date</span>
- Status — <span className="font-normal">current task state</span>
- Assignee — <span className="font-normal">avatar or full name of the user responsible for the task</span>
- Category — <span className="font-normal">group used to organize the task</span>
- Project — <span className="font-normal">project the task belongs to</span>

### Actions

- Clicking on the task title <span className="font-normal">opens a side panel with task details: description, deadline, status, category, assignee, project, and subtask list. This allows you to quickly view full information without navigating to a separate page.</span>
- Clicking on the task assignee name <span className="font-normal">opens a user side panel with position, contacts, a short description, and other details. Helps quickly understand the employee’s role and navigate to their profile if needed.</span>
- Clicking on the project title <span className="font-normal">opens a side panel with project details: description, deadline, status, category, client, and project author. Allows quick access to full information without leaving the page.</span>
- Clicking the "Comments" button <span className="font-normal">opens a modal window with task comments.</span>
- Clicking the "Actions menu" <span className="font-normal">opens a list of available actions.</span>
