## Действия с подзадачами

В системе доступны основные действия, которые помогают работать с подзадачами:

- Добавление подзадачи — <span className="font-normal">создание новой подзадачи внутри основной задачи.</span>
- Редактирование подзадачи — <span className="font-normal">изменение текста подзадачи.</span>
- Удаление подзадачи — <span className="font-normal">удаление подзадачи из списка.</span>
- Изменение статуса — <span className="font-normal">обновление состояния подзадачи (выполнено / невыполнено).</span>

---

### Добавление подзадачи

Чтобы добавить подзадачу, перейдите на страницу детализации задачи, затем к списку подзадач и нажмите кнопку <b>«Добавить»</b> (иконка плюс). Откроется модальное окно, в котором необходимо ввести текст подзадачи.

### Управление подзадачей

Чтобы выполнить действия с подзадачей, нажмите кнопку <b>«Меню действий»</b> (иконка с тремя точками) справа в элементе списка.

#### Редактирование подзадачи

Чтобы редактировать подзадачу в <b>«Меню действий»</b> выберите пункт <b>«Редактировать»</b>. После этого откроется окно редактирования, в котором можно изменить текст подзадачи. Текст подзадачи не может быть пустыми.

#### Удаление подзадачи

Чтобы удалить подзадачу в <b>«Меню действий»</b> выберите пункт <b>«Удалить»</b>. В появившемся модальном окне нажмите кнопку <b>«Удалить»</b>.

#### Изменение статуса

Чтобы изменить статус подзадачи в <b>«Меню действий»</b> выберите пункт:

- «Отметить как выполнено»
- «Отметить как невыполнено»

После выбора статус подзадачи обновится.
