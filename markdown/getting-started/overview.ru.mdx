## Обзор

<b>Taskry</b> предоставляет функционал для управления проектами, задачами,
пользователя и клиентами, а также дополнительными сущностями системы. Ниже
приведен краткий обзор основных разделов приложения.

---

### Общие

В разделе <b>Общие</b> описаны основные функции системы.

- Регистрация — <span className="font-normal">создание аккаунта</span>
- Вход — <span className="font-normal">вход в систему</span>
- Поиск — <span className="font-normal">поиск проектов, задач, пользователей и клиентов</span>
- Язык и тема — <span className="font-normal">переключение языка и темы интерфейса</span>

---

### Проекты

В разделе <b>Проекты</b> описаны основные операции по управлению проектами.

- Создание проектов — <span className="font-normal">добавление новых проектов</span>
- Просмотр проектов — <span className="font-normal">отображение списка или карточек проектов</span>
- Редактирование проектов — <span className="font-normal">изменение данных проекта</span>
- Изменение статуса проектов — <span className="font-normal">обновление текущего состояния проекта</span>
- Удаление проектов — <span className="font-normal">безвозвратное удаление проектов из системы</span>
- Фильтрация и сортировка — <span className="font-normal">фильтрация и сортировка проектов по различным параметрам</span>
- Категории проектов — <span className="font-normal">просмотр, добавление, редактирование и удаление категорий</span>
