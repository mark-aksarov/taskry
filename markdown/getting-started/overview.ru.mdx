## Обзор

<b>Taskry</b> предоставляет функционал для управления проектами и задачами. Ниже
приведен краткий обзор основных разделов приложения.

---

### Проекты

В разделе <b>Проекты</b> описаны основные операции по управлению проектами.

- Создание проектов — <span className="font-normal">добавление новых проектов</span>
- Просмотр проектов — <span className="font-normal">отображение списка или карточек проектов</span>
- Редактирование проектов — <span className="font-normal">изменение данных проекта</span>
- Изменение статуса проектов — <span className="font-normal">обновление текущего состояния проекта</span>
- Удаление проектов — <span className="font-normal">безвозвратное удаление проектов из системы</span>
- Фильтрация и сортировка — <span className="font-normal">поиск и упорядочивание проектов по различным параметрам</span>

---
