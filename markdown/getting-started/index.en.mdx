## Getting started

Welcome to <b>Taskry</b> — a project and task management system designed to efficiently organize team workflows. Taskry helps track projects, manage tasks, interact with clients, and coordinate team members.

---

### Main sections

The documentation consists of the following main sections, each providing tools for managing specific system entities:

- Projects — <span className="font-normal">management of projects, their structure, and related data.</span>
- Tasks — <span className="font-normal">units of work within projects that allow planning and tracking progress.</span>
- Customers — <span className="font-normal">management of clients and their contact information.</span>
- Team — <span className="font-normal">management of system users and their access permissions.</span>
