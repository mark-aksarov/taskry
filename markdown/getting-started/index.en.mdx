## Getting started

<b>Taskry</b> is a project and task management system. <b>Taskry</b> helps you
manage projects, tasks, users, and customers, as well as related data.

---

### Main sections

The documentation includes the following sections. Each one explains tools and features for working with different parts of the system:

- Projects — <span className="font-normal">Manage projects and project categories. Create, edit, delete, filter, and sort.</span>
- Tasks — <span className="font-normal">Manage tasks, subtasks, and task categories. Create, edit, delete, filter, and sort.</span>
- Customers — <span className="font-normal">Manage customers and companies. Create, edit, delete, filter, and sort.</span>
- Team — <span className="font-normal">Manage users and roles. Create, edit, delete, filter, sort, and access permissions.</span>
