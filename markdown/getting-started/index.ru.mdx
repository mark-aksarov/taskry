## Краткое руководство

Добро пожаловать в <b>Taskry</b> — систему управления проектами и задачами, предназначенную для эффективной организации работы команды. Taskry помогает отслеживать проекты, управлять задачами, взаимодействовать с клиентами и координировать работу пользователей.

---

### Основные разделы

Документация состоит из следующих основных разделов, каждый из которых предоставляет инструменты для управления соответствующими сущностями системы:

- Проекты — <span className="font-normal">управление проектами, их структурой и связанными данными.</span>
- Задачи — <span className="font-normal">единицы работы внутри проектов, позволяющие планировать и отслеживать выполнение.</span>
- Клиенты — <span className="font-normal">управление клиентами и их контактной информацией.</span>
- Команда — <span className="font-normal">управление пользователями системы и их права доступа.</span>
