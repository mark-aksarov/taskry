## Overview

<b>Taskry</b> provides functionality for managing projects and tasks. Below is a
brief overview of the main sections of the application.

---

### Projects

The <b>Projects</b> section covers the main operations for managing projects.

- Project creation — <span className="font-normal">adding new projects</span>
- Project viewing — <span className="font-normal">displaying projects as a list or cards</span>
- Project editing — <span className="font-normal">modifying project data</span>
- Project status updates — <span className="font-normal">updating the current state of a project</span>
- Project deletion — <span className="font-normal">permanently removing projects from the system</span>
- Filtering and sorting — <span className="font-normal">searching and organizing projects by various parameters</span>

---
