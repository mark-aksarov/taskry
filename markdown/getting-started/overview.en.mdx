## Overview

<b>Taskry</b> provides features for managing projects, tasks, users, and
customers, as well as other system data. Below is a short overview of the main
sections of the app.

---

### Projects

The <b>Projects</b> section describes the main project management features.

- Create projects — <span className="font-normal">add new projects</span>
- View projects — <span className="font-normal">show project list or cards</span>
- Edit projects — <span className="font-normal">update project data</span>
- Change project status — <span className="font-normal">update current project state</span>
- Delete projects — <span className="font-normal">permanently remove projects from the system</span>
- Filter and sort — <span className="font-normal">filter and sort projects by different parameters</span>
- Project categories — <span className="font-normal">view, add, edit, and delete categories</span>
