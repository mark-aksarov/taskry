## Task filters

To filter tasks, go to the <b>"Tasks"</b> page and click the <b>"Filters"</b> button at the top of the page. A filtering modal window will open, where the following filters are available:

- Only my tasks — <span className="font-normal">displays tasks assigned to the current user.</span>
- Deadline from / to — <span className="font-normal">allows selecting a date range.</span>
- Status — <span className="font-normal">"Pending", "Active", "Completed".</span>
- Category — <span className="font-normal">for example: "Design", "Internal Systems", "Web Development".</span>
- Project — <span className="font-normal">filters tasks by the selected project.</span>
- Assignee — <span className="font-normal">filters tasks by the assigned user.</span>
