# Поиск и фильтрация

Здесь объясняется, как пользователь может находить нужные задачи среди большого количества записей, а также <b>фильтрация</b> и <b>сортировка</b>, которые помогают сузить выборку.

---

## Фильтрация задач

Чтобы отфильтровать задачи, перейдите на страницу <b>Задачи</b>, нажмите кнопку <b>Фильтры</b> в верхней части страницы. Откроется модальное окно фильтрации, где доступны следующие фильтры:

- Только мои задачи — <span className="font-normal">отображение задач, назначенных текущему пользователю.</span>
- Дедлайн с / по — <span className="font-normal">позволяет выбрать диапазон дат.</span>
- Статус — <span className="font-normal">фильтрация по состоянию задачи: <b>В ожидании</b>, <b>Активный</b>, <b>Завершён</b>.</span>
- Категория — <span className="font-normal">позволяет фильтровать и группировать задачи по категориям.</span>
- Проект — <span className="font-normal">фильтрация задач по выбранному проекту.</span>
- Исполнитель — <span className="font-normal">фильтрация задач по назначенным пользователям.</span>

## Сортировка задач

Чтобы отсортировать задачи, перейдите на страницу <b>Задачи</b>, нажмите кнопку <b>Сортировка</b> в верхней части страницы. В открывшемся меню доступны следующие варианты сортировки:

- По дате создания — <span className="font-normal">сортировка от новых к старым.</span>
- По заголовку — <span className="font-normal">сортировка в алфавитном порядке.</span>
- По сроку — <span className="font-normal">сортировка по ближайшему дедлайну.</span>
- По статусу — <span className="font-normal">сортировка по состоянию проекта: <b>Активный</b>, <b>В ожидании</b>, <b>Завершён</b>.</span>
