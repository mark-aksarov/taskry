# Действия с задачами

В этом разделе описаны основные действия с задачами: создание, редактирование, изменение статуса и удаление.

---

## Создание задачи

Для создания задачи на странице <b>Задачи</b> нужно нажать кнопку <b>Создать новую</b>, выбрать пункт <b>Задача</b>, заполнить необходимые поля в форме и подтвердить создание задачи.

Создание задачи включает заполнение следующих данных:

- Заголовок — <span className="font-normal">краткое название задачи</span>
- Срок выполнения — <span className="font-normal">срок завершения задачи</span>
- Статус — <span className="font-normal">текущее состояние задачи</span>
- Описание — <span className="font-normal">дополнительная информация о задаче</span>
- Категория — <span className="font-normal">группа для организации задачи</span>
- Проект — <span className="font-normal">проект, к которому относится задача</span>
- Исполнитель — <span className="font-normal">пользователь, ответственный за выполнение задачи</span>

Поля <b>Заголовок</b> и <b>Срок выполнения</b> являются обязательными для заполнения. 

После сохранения задача будет успешно создана.

## Удаление задачи

В системе предусмотрено два типа удаления: <b>одиночное</b> и <b>массовое</b>. На мобильных устройствах доступно только <b>одиночное</b> удаление. Оба варианта позволяют безвозвратно удалить задачи из системы.
Удаление задачи является необратимым действием, поэтому перед удалением рекомендуется убедиться, что задача больше не нужна и все важные данные сохранены.

### Одиночное удаление

Чтобы удалить задачу на странице <b>Задачи</b> нажмите кнопку <b>Меню действий</b> (иконка с тремя точками) в карточке или списке задач и выберите пункт <b>Удалить</b>.
