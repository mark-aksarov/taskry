## Subtask actions

The system provides basic actions that help you work with subtasks:

- Add subtask — <span className="font-normal">create a new subtask inside the main task.</span>
- Edit subtask — <span className="font-normal">modify the subtask text.</span>
- Delete subtask — <span className="font-normal">remove a subtask from the list.</span>
- Change status — <span className="font-normal">update the subtask state (done / undone).</span>

---

### Adding a subtask

To add a subtask, go to the task details page, then scroll to the subtasks list and click the <b>“Add”</b> button (plus icon). A modal window will open where you need to enter the subtask text.

#### Editing a subtask

To edit a subtask, select <b>“Edit”</b> from the <b>“Actions menu”</b>. A modal window will open where you can change the subtask text. The subtask text cannot be empty.

#### Changing status

To change the status of a subtask, select one of the options in the <b>“Actions menu”</b>:

- “Mark as done”
- “Mark as undone”

After selection, the subtask status will be updated.
