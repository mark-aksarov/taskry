## Задачи

Задача — это единица работы, которая содержит данные, необходимые для выполнения конкретного действия в рамках рабочего процесса:

- Заголовок — <span className="font-normal">краткое название задачи</span>
- Описание — <span className="font-normal">подробная информация о задаче</span>
- Дедлайн — <span className="font-normal">срок, к которому задача должна быть выполнена</span>
- Проект — <span className="font-normal">проект, к которому относится задача</span>
- Исполнитель — <span className="font-normal">пользователь, ответственный за выполнение задачи</span>
- Автор — <span className="font-normal">пользователь, создавший задачу</span>
- Категория — <span className="font-normal">логическая группировка задачи</span>
- Статус — <span className="font-normal">текущее состояние задачи</span>
- Подзадачи — <span className="font-normal">разделение задачи на более мелкие части</span>
- Комментарии — <span className="font-normal">обсуждения и заметки по задаче</span>

---

### Просмотр и навигация

Задачи отображаются в виде списка или сетки на странице <b>«Задачи»</b>. Чтобы перейти в этот раздел, необходимо выбрать пункт <b>«Задачи»</b> в боковой навигации приложения.

Также просмотр и редактирование задачи доступны на странице детализации задачи. Для этого необходимо нажать на заголовок задачи и в боковой панели нажать кнопку <b>«Открыть задачу»</b>. На этой странице можно изменять каждое поле в отдельном модальном окне.

---

### Действия

Для задач доступны следующие основные действия:

- Создание задачи — <span className="font-normal">добавление новой задачи с заполнением базовой информации: название, дедлайн, исполнитель, статус, категория и другие параметры</span>
- Редактирование задачи — <span className="font-normal">изменение данных существующей задачи</span>
- Удаление задачи — <span className="font-normal">удаление одной или нескольких задач из системы</span>
- Изменение статуса — <span className="font-normal">обновление текущего состояния задачи</span>

Работа с подзадачами включает следующие действия:

- Добавление подзадачи — <span className="font-normal">создание новой подзадачи внутри задачи</span>
- Редактирование подзадачи — <span className="font-normal">изменение данных существующей подзадачи</span>
- Переключение статуса подзадачи — <span className="font-normal">изменение состояния подзадачи (выполнено / невыполнено)</span>
- Удаление подзадачи — <span className="font-normal">удаление подзадачи из задачи</span>

Работа с категориями задач включает следующие действия:

- Создание категории задач — <span className="font-normal">добавление новой категории задач</span>
- Редактирование категории задач — <span className="font-normal">изменение названия существующей категории</span>
- Удаление категории задач — <span className="font-normal">удаление категории из системы</span>
