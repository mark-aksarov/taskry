# Категории задач

Категория задачи — это способ группировки задач по типу или тематике. Каждая категория имеет уникальное название. Задача может быть назначена только в одну категорию.

---

## Просмотр категорий

Категории задач отображаются в выпадающих списках при создании или редактировании задачи. Также они используются при фильтрации.

Чтобы просмотреть все категории, перейдите на страницу <b>Задачи</b>, нажмите кнопку <b>Управление</b> в верхней части страницы и выберите пункт меню <b>Категории</b>, чтобы перейти на страницу <b>Категории задач</b>.

## Действия с категориями

Работа с категориями задач включает следующие действия:

- Создание категории задачи — <span className="font-normal">добавление новой категории задачи</span>
- Редактирование категории задачи — <span className="font-normal">изменение названия существующей категории</span>
- Удаление категории задачи — <span className="font-normal">удаление категории из системы</span>

---

### Добавление категории

Чтобы добавить категорию на странице <b>Категории задач</b>, нажмите кнопку <b>Создать новую</b> в верхней части страницы. В появившемся модальном окне введите название категории задачи и подтвердите создание категории.

### Удаление категории

Чтобы удалить категорию в <b>Меню действий</b> выберите пункт <b>Удалить</b>. Вместе с категорией будут удалены все связанные задачи. Удаление категории является необратимым действием, поэтому перед удалением рекомендуется убедиться, что категория больше не нужна и все важные данные сохранены.
