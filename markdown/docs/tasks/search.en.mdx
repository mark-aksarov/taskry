# Search and filtering

This section explains how users can find the required tasks among many records, as well as use <b>filtering</b> and <b>sorting</b> to narrow down the results.

---

## Filtering tasks

To filter tasks, go to the <b>Tasks</b> page and click the <b>Filters</b> button at the top of the page. A filter dialog will open with the following options:

- My tasks only — <span className="font-normal">shows only tasks assigned to the current user.</span>
- Deadline from / to — <span className="font-normal">allows selecting a date range.</span>
- Status — <span className="font-normal">filters tasks by status: <b>Pending</b>, <b>Active</b>, <b>Completed</b>.</span>
- Category — <span className="font-normal">allows filtering and grouping tasks by category.</span>
- Project — <span className="font-normal">filters tasks by the selected project.</span>
- Assignee — <span className="font-normal">filters tasks by assigned users.</span>

## Sorting tasks

To sort tasks, go to the <b>Tasks</b> page and click the <b>Sorting</b> button at the top of the page. The following sorting options are available:

- By creation date — <span className="font-normal">sorts tasks from newest to oldest.</span>
- By title — <span className="font-normal">sorts tasks alphabetically.</span>
- By deadline — <span className="font-normal">sorts tasks by the nearest deadline.</span>
- By status — <span className="font-normal">sorts tasks by status: <b>Active</b>, <b>Pending</b>, <b>Completed</b>.</span>
