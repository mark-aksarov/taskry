# Subtasks

Subtasks allow you to split a main task into smaller parts for easier management and progress tracking.

Each subtask contains the following information:

- Text — <span className="font-normal">a short description of the subtask.</span>
- Status — <span className="font-normal">shows whether the subtask is completed or not.</span>

---

## Subtask actions

The system provides the following actions for working with subtasks:

- Add subtask — <span className="font-normal">create a new subtask inside the main task.</span>
- Edit subtask — <span className="font-normal">change the subtask text.</span>
- Delete subtask — <span className="font-normal">remove a subtask from the list.</span>
- Change status — <span className="font-normal">update the subtask status (completed / not completed).</span>

---
