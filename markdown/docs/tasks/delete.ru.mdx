## Удаление задачи

В системе предусмотрено два типа удаления: <b>одиночное</b> и <b>массовое</b>. Оба варианта позволяют безвозвратно удалить задачи из системы.
Удаление задач является необратимым действием, поэтому перед удалением рекомендуется убедиться, что задача больше не нужна и все важные данные сохранены.

---

### Одиночное удаление

Чтобы удалить задачу, перейдите на страницу <b>«Задачи»</b>, нажмите кнопку <b>«Меню действий»</b> (иконка с тремя точками) справа в карточке или в элементе списка и в появившемся меню выберите пункт <b>«Удалить»</b>.

### Массовое удаление

Чтобы удалить несколько задач одновременно, отметьте нужные задачи с помощью чекбоксов. Затем нажмите кнопку <b>«Действия»</b> в верхней части страницы и выберите пункт <b>«Удалить»</b>.
