# Task actions

This section describes the main task actions: creating, editing, changing status, and deleting tasks.

---

## Creating a task

To create a task on the <b>Tasks</b> page, click the <b>Create new</b> button, select <b>Task</b>, fill in the required fields in the form, and confirm task creation.

Creating a task includes filling in the following information:

- Title — <span className="font-normal">a short name of the task</span>
- Deadline — <span className="font-normal">the task completion deadline</span>
- Status — <span className="font-normal">the current state of the task</span>
- Description — <span className="font-normal">additional information about the task</span>
- Category — <span className="font-normal">a group used to organize the task</span>
- Project — <span className="font-normal">the project the task belongs to</span>
- Assignee — <span className="font-normal">the user responsible for completing the task</span>

The <b>Title</b> and <b>Deadline</b> fields are required.

After saving, the task will be created successfully.

## Deleting a task

The system supports two types of deletion: <b>single deletion</b> and <b>bulk deletion</b>. On mobile devices, only <b>single deletion</b> is available. Both options permanently remove tasks from the system.

Deleting a task cannot be undone. Before deleting a task, make sure it is no longer needed and that any important information has been saved.

### Single deletion

To delete a task on the <b>Tasks</b> page, click the <b>Actions menu</b> button (the three-dot icon) in the task card or task list and select <b>Delete</b>.

## Changing status

The system supports two ways to change status: <b>single status change</b> and <b>bulk status change</b>. On mobile devices, only <b>single status change</b> is available.

### Single status change

To change the status of a task on the <b>Tasks</b> page, click the <b>Actions Menu</b> button (the three-dot icon) in the task card or task list and select a new status.
