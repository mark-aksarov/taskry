## Task categories

A task category is a way to group tasks by type or topic. Each category has a unique name. A task can only be assigned to one category.

---

### Viewing categories

Task categories are shown in dropdown lists when you create or edit a task. They are also available on the categories page and used for filtering.

To view all categories, go to the <b>“Tasks”</b> page, click the <b>“Manage”</b> button at the top of the page, and select <b>“Categories”</b> from the menu to open the <b>“Task categories”</b> page.
