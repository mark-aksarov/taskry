# Task categories

A task category is a way to group tasks by type or topic. Each category has a unique name. A task can belong to only one category.

---

## Viewing categories

Task categories are displayed in dropdown lists when creating or editing a task. They are also used for filtering.

To view all categories, go to the <b>Tasks</b> page, click the <b>Manage</b> button at the top of the page, and select <b>Categories</b> to open the <b>Task categories</b> page.

## Category actions

Working with task categories includes the following actions:

- Create category — <span className="font-normal">add a new task category.</span>
- Edit category — <span className="font-normal">change the name of an existing category.</span>
- Delete category — <span className="font-normal">remove a category from the system.</span>

---

### Adding a category

To add a category on the <b>Task categories</b> page, click the <b>Create new</b> button at the top of the page. In the dialog that appears, enter the category name and confirm creation.

### Deleting a category

To delete a category, open the <b>Actions menu</b> and select <b>Delete</b>. All tasks linked to this category will also be deleted. Deleting a category cannot be undone, so make sure the category is no longer needed and that any important information has been saved.
