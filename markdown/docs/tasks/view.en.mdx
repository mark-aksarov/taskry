# Viewing tasks

Tasks are displayed on the <b>Tasks</b> page as a list or as cards. On mobile devices, only the card view is available.

A task in the list or card view may contain the following information (depending on the view type):

- Title — <span className="font-normal">a short name of the task</span>
- Deadline — <span className="font-normal">the task completion deadline</span>
- Status — <span className="font-normal">the current state of the task</span>
- Assignee — <span className="font-normal">the image or full name of the user responsible for completing the task</span>
- Category — <span className="font-normal">a group used to organize the task</span>
- Project — <span className="font-normal">the project the task belongs to</span>
- Progress — <span className="font-normal">a visual indicator showing the completion of the task's subtasks</span>

On desktop devices, clicking the <b>task title</b>, <b>project title</b>, <b>author name</b>, or <b>client name</b> opens a side panel with related information about the task, project, user, or client.

On mobile devices, the behavior is simplified: tapping a task card opens the task details page directly.

The <b>Comments</b> button opens the task comments modal, while the <b>Actions menu</b> provides access to additional actions.
