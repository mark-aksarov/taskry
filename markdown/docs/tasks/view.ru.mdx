# Просмотр задач

Задачи отображаются на странице <b>Задачи</b> в виде списка или карточек. На мобильных устройствах доступен только режим карточек.

Задача в списке и карточке может содержать следующие данные (в зависимости от типа отображения):

- Заголовок — <span className="font-normal">краткое название задачи</span>
- Срок выполнения — <span className="font-normal">срок завершения задачи</span>
- Статус — <span className="font-normal">текущее состояние задачи</span>
- Исполнитель — <span className="font-normal">изображение или полное имя пользователя, ответственного за выполнение задачи</span>
- Категория — <span className="font-normal">группа для организации задачи</span>
- Проект — <span className="font-normal">проект, к которому относится задача</span>
- Прогресс — <span className="font-normal">отображение выполнения подзадач задачи в визуальной шкале</span>

На десктопных устройствах при нажатии на <b>заголовок задачи</b>, <b>заголовок проекта</b>, <b>имя автора</b> или <b>имя клиента</b> открывается боковая панель с соответствующей информацией (о задаче, проекте, пользователе или клиенте). 

На мобильных устройствах поведение упрощено: нажатие на карточку сразу выполняет переход на страницу детализации задачи.

Отдельная кнопка <b>комментарии</b> открывает окно с комментариями задачи, а <b>меню действий</b> предоставляет доступ к дополнительным операциям.
