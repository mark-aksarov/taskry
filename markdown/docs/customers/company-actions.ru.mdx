## Действия с компаниями

Работа с компаниями включает следующие действия:

- Создание компании — <span className="font-normal">добавление новой компании</span>
- Редактирование компании — <span className="font-normal">изменение названия существующей компании</span>
- Удаление компании — <span className="font-normal">удаление компании из системы</span>

---

### Добавление компании

Чтобы добавить компанию, перейдите на страницу <b>«Компании»</b>, нажмите кнопку <b>«Создать новую»</b> в верхней части страницы, в появившемся модальном окне введите название компании.

### Управление компанией

Чтобы выполнить действия с существующей компанией, нажмите кнопку <b>«Меню действий»</b> (иконка с тремя точками) справа в элементе списка.

#### Редактирование компании

Чтобы отредактировать компанию в <b>«Меню действий»</b> выберите пункт <b>«Редактировать»</b>. После этого откроется окно редактирования, в котором можно изменить название компании. Название компании не может быть пустыми.

#### Удаление компании

Чтобы удалить компанию в <b>«Меню действий»</b> выберите пункт <b>«Удалить»</b>. В появившемся модальном окне нажмите кнопку <b>«Удалить»</b>.
