## Сортировка клиентов

Чтобы отсортировать клиентов, перейдите на страницу <b>«Клиенты»</b>, нажмите кнопку <b>«Сортировка»</b> в верхней части страницы. В открывшемся меню доступны следующие варианты сортировки:

- По полному имени — <span className="font-normal">сортирует клиентов по полному имени.</span>
- По компании — <span className="font-normal">сортирует клиентов по компании.</span>
