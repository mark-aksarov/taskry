# Поиск и фильтрация

Здесь объясняется, как пользователь может находить клиентов, а также <b>фильтрация</b> и <b>сортировка</b>.

## Фильтрация клиентов

Чтобы отфильтровать клиентов, перейдите на страницу <b>Клиенты</b>, нажмите кнопку <b>Фильтры</b> в верхней части страницы. Откроется модальное окно фильтрации, где доступны следующие фильтры:

- Не имеет активных проектов — <span className="font-normal">отображает только клиентов без активных проектов.</span>
- Имеет активные проекты — <span className="font-normal">отображает только клиентов, у которых есть активные проекты.</span>
- Имеет просроченные проекты — <span className="font-normal">отображает только клиентов с просроченными проектами.</span>
- Компания — <span className="font-normal">фильтрация клиентов по компании.</span>

## Сортировка клиентов

Чтобы отсортировать клиентов, перейдите на страницу <b>Клиенты</b>, нажмите кнопку <b>Сортировка</b> в верхней части страницы. В открывшемся меню доступны следующие варианты сортировки:

- По полному имени — <span className="font-normal">сортирует клиентов по полному имени.</span>
- По компании — <span className="font-normal">сортирует клиентов по компании.</span>
