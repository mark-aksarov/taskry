# Viewing customer

Customers are displayed on the <b>Customers</b> page in either a list or card view. On mobile devices, only the card view is available.

A customer item in the list or card may include the following data (depending on the view type):

- Full name — <span className="font-normal">the displayed name of the customer.</span>
- Avatar — <span className="font-normal">customer image.</span>
- Email — <span className="font-normal">email address used to contact the customer.</span>
- Phone number — <span className="font-normal">customer contact phone number.</span>
- Public link — <span className="font-normal">link to the customer profile.</span>
- Company — <span className="font-normal">company associated with the customer.</span>

On desktop devices, clicking a customer’s <b>name</b> opens a side panel with detailed customer information.

On mobile devices, behavior is simplified: tapping a card immediately navigates to the customer profile page.

A dedicated <b>actions menu</b> button provides access to additional operations.
