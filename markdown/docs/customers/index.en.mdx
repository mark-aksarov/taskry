# Customers

A customer is an entity that contains information about a customer:

- Full Name — <span className="font-normal">the customer's display name.</span>
- About Me — <span className="font-normal">short information about the customer shown on the profile page.</span>
- Email — <span className="font-normal">email address used to contact the customer.</span>
- Phone Number — <span className="font-normal">the customer's phone number.</span>
- Public Link — <span className="font-normal">a link to the customer's profile.</span>
- Company — <span className="font-normal">the company associated with the customer.</span>

---

## Overview

This section explains how to work with customers and the main features available to users in the system.

### Viewing customers

This section describes how customers are displayed in the system interface. It covers two view modes—list and cards—and the summary information available for each customer.

### Customer actions

This section covers the main customer actions: creating, editing, and deleting customers.

### Search and filtering

This section explains how users can find customers, as well as filtering and sorting options that help narrow the results.

### Companies

This section describes companies and the actions available to users, including viewing the company list, creating companies, editing companies, and deleting companies. A company represents an organization associated with a customer and is used to group customers.
