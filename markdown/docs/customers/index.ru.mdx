# Клиенты

Клиент — это единица работы, которая содержит данные, необходимые для выполнения конкретного действия:

- Полное имя — <span className="font-normal">отображаемое имя клиента.</span>
- О себе - <span className="font-normal">краткая информация о клиенте, отображаемая в профиле.</span>
- Email — <span className="font-normal">электронная почта для связи с клиентом.</span>
- Номер телефона — <span className="font-normal">номер телефона клиента.</span>
- Публичная ссылка — <span className="font-normal">ссылка на профиль клиента.</span>
- Компания — <span className="font-normal">компания, с которой связан клиент.</span>

---

## Краткий обзор

В этом разделе рассматривается работа с клиентами и основные возможности, которые доступны пользователю в системе.

### Просмотр клиентов

Здесь описывается, как задачи отображаются в интерфейсе системы. Рассматриваются два формата представления — список и карточки, а также то, какую краткую информацию пользователь может увидеть о задаче.

### Действия с клиентами

Здесь рассматриваются основные действия с клиентами: создание новых клиентов, редактирование и удаление клиентов.

### Поиск и фильтрация

Здесь объясняется, как пользователь может находить нужных клиентов, а также фильтрация и сортировка, которые помогают сузить выборку.

### Компании

Здесь рассматриваются компании, а также какие действия доступны пользователю — просмотр списка компании, их создание, редактирование и удаление. Компания представляет организацию, связанную с клиентом, и используется для группировки клиентов.
