## Клиенты

Клиент — это сущность, которая представляет контакт, участвующий в проектах внутри системы. Используется для хранения контактной информации и связи с проектами и компанией.

Хранит данные:

- Полное имя — <span className="font-normal">отображаемое имя клиента.</span>
- О себе - <span className="font-normal">краткая информация о клиенте, отображаемая в профиле.</span>
- Email — <span className="font-normal">электронная почта для связи с клиентом.</span>
- Номер телефона — <span className="font-normal">номер телефона клиента.</span>
- Публичная ссылка — <span className="font-normal">ссылка на профиль клиента.</span>
- Компания — <span className="font-normal">компания, с которой связан клиент.</span>

---

### Просмотр и навигация

Клиенты отображаются в виде списка или сетки на странице <b>«Клиенты»</b>. Чтобы перейти в этот раздел, необходимо выбрать пункт <b>«Клиенты»</b> в боковой навигации приложения.

Также просмотр и редактирование клиента доступно на странице профиля клиента. Для этого необходимо нажать на имя клиента и в боковой панели выбрать действие <b>«Открыть клиента»</b>. На этой странице доступно редактирование каждого поля через отдельные модальные окна.

---

### Действия

С клиентами доступны следующие основные действия:

- Создание клиента — <span className="font-normal">добавление новой записи клиента с заполнением основных данных: полное имя, email, телефон, о себе, компания</span>
- Редактирование клиента — <span className="font-normal">изменение существующих данных клиента</span>
- Удаление клиента — <span className="font-normal">удаление одного или нескольких клиентов вместе со связанными данными</span>
