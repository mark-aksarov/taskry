# Просмотр клиентов

Клиенты отображаются на странице <b>Клиенты</b> в виде списка или карточек. На мобильных устройствах доступен только режим карточек.

Клиент в списке и карточке может содержать следующие данные (в зависимости от типа отображения):

- Полное имя — <span className="font-normal">отображаемое имя клиента.</span>
- Аватар — <span className="font-normal">изображение клиента.</span>
- Email — <span className="font-normal">электронная почта для связи с клиентом.</span>
- Номер телефона — <span className="font-normal">номер телефона клиента.</span>
- Публичная ссылка — <span className="font-normal">ссылка на профиль клиента.</span>
- Компания — <span className="font-normal">компания, с которой связан клиент.</span>

На десктопных устройствах при нажатии на <b>имя клиента</b> открывается боковая панель с соответствующей информацией о клиенте. 

На мобильных устройствах поведение упрощено: нажатие на карточку сразу выполняет переход на страницу профиля клиента.

Отдельная кнопка <b>меню действий</b> предоставляет доступ к дополнительным операциям.
