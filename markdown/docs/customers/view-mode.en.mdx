## Client view

The system supports two display modes for clients. In <b>“List”</b> mode, data is shown as list items. In <b>“Cards”</b> mode, clients are displayed in a grid, where each client is represented as a separate card. Both modes use the same dataset but differ in visual presentation.

---

### List

To switch to list mode, go to the <b>“Clients”</b> page and click the <b>“List”</b> button in the view toggle panel at the top of the page.

### Actions

- Clicking on a client’s name or image <span className="font-normal">opens a side panel with client details: contact information, company, short description, and other data. It allows viewing client information and navigating to their profile if needed.</span>
- Clicking on email <span className="font-normal">opens the email client via a mailto link to send an email.</span>
- Clicking on the phone number <span className="font-normal">initiates a call via a tel link on supported devices.</span>
- Clicking on the public link <span className="font-normal">opens the client profile page in a new tab.</span>
- Clicking on “Actions menu” <span className="font-normal">opens a list of available actions.</span>
