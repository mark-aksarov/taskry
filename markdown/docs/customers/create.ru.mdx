## Создание клиента

Чтобы создать клиента, перейдите на страницу <b>«Клиента»</b>, нажмите кнопку <b>«Создать нового»</b> в верхней части страницы и в появившемся меню выберите пункт <b>«Клиент»</b>.

Создание клиента включает заполнение следующих данных:

- Полное имя — <span className="font-normal">отображаемое имя клиента.</span>
- О себе - <span className="font-normal">краткая информация о клиенте, отображаемая в профиле.</span>
- Email — <span className="font-normal">электронная почта для связи с клиентом.</span>
- Номер телефона - <span className="font-normal">контактный номер клиента для связи.</span>
- Публичная ссылка - <span className="font-normal">ссылка на профиль клиента.</span>
- Адрес - <span className="font-normal">местоположение или адрес клиента.</span>
- Компания — <span className="font-normal">компания, с которой связан клиент.</span>

После сохранения в системе клиент появится в списке клиентов, если он соответствует текущей сортировке, фильтрам и условиям поиска.
