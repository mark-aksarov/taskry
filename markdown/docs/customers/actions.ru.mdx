# Действия с клиентами

В этом разделе описаны основные действия с клиентами: создание, редактирование и удаление.

---

## Создание клиента

Для создания клиента на странице <b>Клиенты</b> нужно нажать кнопку <b>Создать нового</b>, выбрать пункт <b>Клиент</b>, заполнить необходимые поля в форме и подтвердить создание клиента.

Создание клиента включает заполнение следующих данных:

- Полное имя — <span className="font-normal">отображаемое имя клиента.</span>
- О себе - <span className="font-normal">краткая информация о клиенте, отображаемая в профиле.</span>
- Email — <span className="font-normal">электронная почта для связи с клиентом.</span>
- Номер телефона - <span className="font-normal">контактный номер клиента для связи.</span>
- Публичная ссылка - <span className="font-normal">ссылка на профиль клиента.</span>
- Адрес - <span className="font-normal">местоположение или адрес клиента.</span>
- Компания — <span className="font-normal">компания, с которой связан клиент.</span>

После сохранения в системе клиент появится в списке клиентов, если он соответствует текущей сортировке, фильтрам и условиям поиска.

## Удаление клиента

В системе предусмотрено два типа удаления: <b>одиночное</b> и <b>массовое</b>. На мобильных устройствах доступно только <b>одиночное</b> удаление. Оба варианта позволяют безвозвратно удалить клиента из системы.
Удаление клиента является необратимым действием, поэтому перед удалением рекомендуется убедиться, что задача больше не нужна и все важные данные сохранены.

### Одиночное удаление

Чтобы удалить клиента на странице <b>Клиенты</b> нажмите кнопку <b>Меню действий</b> (иконка с тремя точками) в карточке или списке задач и выберите пункт <b>Удалить</b>.
