# Customer actions

This section describes the main customer operations: creation, editing, and deletion.

---

## Create customer

To create a customer on the <b>Customers</b> page, click the <b>Create New</b> button, select <b>Customer</b>, fill in the required fields in the form, and confirm creation.

Customer creation includes filling in the following data:

- Full name — <span className="font-normal">the displayed name of the customer.</span>
- About — <span className="font-normal">a short description shown in the customer profile.</span>
- Email — <span className="font-normal">email address used to contact the customer.</span>
- Phone number — <span className="font-normal">customer contact phone number.</span>
- Public link — <span className="font-normal">link to the customer profile.</span>
- Address — <span className="font-normal">customer location or address.</span>
- Company — <span className="font-normal">company associated with the customer.</span>

After saving, the customer will appear in the list if it matches the current sorting, filters, and search conditions.

## Delete customer

The system supports two types of deletion: <b>single</b> and <b>bulk</b>. On mobile devices, only <b>single</b> deletion is available. Both options permanently remove a customer from the system.

Deleting a customer is irreversible, so make sure the customer is no longer needed and all important data is preserved before proceeding.

### Single deletion

To delete a customer on the <b>Customers</b> page, click the <b>Actions menu</b> (three dots icon) in the customer card or list item and select <b>Delete</b>.
