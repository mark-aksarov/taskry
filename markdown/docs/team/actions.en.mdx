# User actions

This section describes the main user actions: creating, editing, and deleting users.

---

## Creating a user

To create a user, go to the <b>Users</b> page, click <b>Create new</b>, select <b>User</b>, fill in the <b>Full Name</b> and <b>Email</b> fields, and confirm the creation.

Editing a user includes the following fields:

- Full Name — <span className="font-normal">the user's display name.</span>
- About Me — <span className="font-normal">short information shown on the user's profile.</span>
- Date of Birth — <span className="font-normal">used to store the user's birth date.</span>
- Phone Number — <span className="font-normal">the user's contact phone number.</span>
- Public Link — <span className="font-normal">a unique profile link available to other users.</span>
- Address — <span className="font-normal">the user's location or address.</span>
- Position — <span className="font-normal">the user's position in the team.</span>

After saving, the changes will immediately appear in the user list.

## Deleting a user

Only <b>single</b> user deletion is available in the system. Deleting a user permanently removes the user from the system.

To delete a user, go to the <b>Team</b> page, click <b>Action menu</b> (the three-dot icon) on a card or list item, and select <b>Delete</b>.
