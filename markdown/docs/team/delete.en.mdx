## User deletion

The system supports only <b>single</b> user deletion. Deletion allows you to permanently remove a user from the system.  
To delete a user, go to the <b>“Team”</b> page, click the <b>“Action menu”</b> button (three dots icon) on the right side of the card or list item, and select <b>“Delete”</b> from the dropdown menu.

In the modal window that appears, click the <b>“Delete”</b> button. The user, along with all associated comments, will be removed. Related tasks and projects will not be deleted, but their <b>“Creator”</b> and <b>“Assignee”</b> fields will be cleared.

After confirmation, the user will be permanently deleted and removed from the user list.
