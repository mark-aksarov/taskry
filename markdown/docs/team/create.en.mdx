## Creating a user

To create a user, go to the <b>“Team”</b> page, click the <b>“Create new”</b> button at the top of the page, and select <b>“User”</b> from the dropdown menu.

After that, a user creation modal will open where you need to fill in <b>“Full name”</b> and <b>“Email”</b>.
