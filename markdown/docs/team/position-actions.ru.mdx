## Действия с должностями

Работа с должностями включает следующие действия:

- Создание должности — <span className="font-normal">добавление новой должности</span>
- Редактирование должности — <span className="font-normal">изменение названия существующей должности</span>
- Удаление должности — <span className="font-normal">удаление должности из системы</span>

---

### Добавление должности

Чтобы добавить должность, перейдите на страницу <b>«Должности»</b>, нажмите кнопку <b>«Создать новую»</b> в верхней части страницы, в появившемся модальном окне введите название должности.

### Управление должностями

Чтобы выполнить действия с существующей должностью, нажмите кнопку <b>«Меню действий»</b> (иконка с тремя точками) справа в элементе списка.

#### Редактирование должности

Чтобы отредактировать должность в <b>«Меню действий»</b> выберите пункт <b>«Редактировать»</b>. После этого откроется окно редактирования, в котором можно изменить название должности. Название должности не может быть пустыми.

#### Удаление должности

Чтобы удалить должность в <b>«Меню действий»</b> выберите пункт <b>«Удалить»</b>. В появившемся модальном окне нажмите кнопку <b>«Удалить»</b>.
