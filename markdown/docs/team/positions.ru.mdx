# Должность

Должность — это сущность, которая определяет должность пользователя и используется для его описания.

---

## Просмотр должностей

Должности отображаются в выпадающем списке при редактировании пользователя. Также они доступны на странице должностей и используются при фильтрации.

Чтобы просмотреть все должности, перейдите на страницу <b>Должности</b>, нажмите кнопку <b>Управление</b> в верхней части страницы и выберите пункт меню <b>Должности</b>, чтобы перейти на страницу <b>Должности</b>.

## Действия с должностями

Работа с должностями включает следующие действия:

- Создание должности — <span className="font-normal">добавление новой должности</span>
- Редактирование должности — <span className="font-normal">изменение названия существующей должности</span>
- Удаление должности — <span className="font-normal">удаление должности из системы</span>

---

### Добавление должности

Чтобы добавить должность, перейдите на страницу <b>Должности</b>, нажмите кнопку <b>Создать новую</b> в верхней части страницы, в появившемся модальном окне введите название должности.
