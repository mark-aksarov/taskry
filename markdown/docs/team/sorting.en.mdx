## User sorting

To sort users, go to the <b>"Team"</b> page and click the <b>"Sort"</b> button at the top. In the menu, you can choose:

- By full name — <span className="font-normal">sorts users by full name.</span>
- By position — <span className="font-normal">sorts users by position.</span>
