## Сортировка пользователей

Чтобы отсортировать пользователей, перейдите на страницу <b>«Команда»</b>, нажмите кнопку <b>«Сортировка»</b> в верхней части страницы. В открывшемся меню доступны следующие варианты сортировки:

- По полному имени — <span className="font-normal">сортирует пользователей по полному имени.</span>
- По должности — <span className="font-normal">сортирует пользователей по должности.</span>
