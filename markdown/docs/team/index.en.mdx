# Team

The team is a list of users within a workspace who work on tasks and projects.

Users are not organized into separate groups. All collaboration happens through system entities:

- Tasks — <span className="font-normal">a user can be an assignee or a creator.</span>
- Projects — <span className="font-normal">a user can be the creator of a project.</span>
- Comments — <span className="font-normal">a user acts as the author of a comment.</span>

A task can have only one assignee.

Each user contains the following information:

- Full Name — <span className="font-normal">the user's display name.</span>
- Email — <span className="font-normal">used for sign-in and communication.</span>
- Role — <span className="font-normal">defines the access level (owner, user, guest).</span>
- Position — <span className="font-normal">the user's position in the team.</span>
- Contact Information — <span className="font-normal">phone number, address, and other fields.</span>
- Email Verified — <span className="font-normal">whether the user has confirmed their email through the verification link.</span>
