# Viewing team

Users are displayed on the <b>Team</b> page as a list or cards. On mobile devices, only the card view is available.

A user in the list or card view may contain the following information (depending on the display type):

- Full Name — <span className="font-normal">the user's display name.</span>
- Avatar — <span className="font-normal">the user's profile image.</span>
- Email — <span className="font-normal">used for communication and opening an email client.</span>
- Phone Number — <span className="font-normal">the user's contact phone number.</span>
- Public Link — <span className="font-normal">a link to the user's public profile.</span>
- Position — <span className="font-normal">the user's role or position in the team.</span>

On desktop devices, clicking the <b>user's name</b> opens a side panel with information about that user.

On mobile devices, the behavior is simplified: tapping a card opens the user's profile page.

A separate <b>Actions menu</b> button provides access to additional operations.
