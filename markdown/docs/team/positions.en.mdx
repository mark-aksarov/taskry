# Position

A position is an entity that defines a user's job role and is used to describe the user.

---

## Viewing positions

Positions are shown in dropdown lists when editing a user. They are also available on the Positions page and used for filtering.

To view all positions, go to the <b>Positions</b> page, click <b>Manage</b> at the top of the page, and select <b>Positions</b> from the menu to open the <b>Positions</b> page.

## Position actions

Working with positions includes the following actions:

- Create position — <span className="font-normal">add a new position</span>
- Edit position — <span className="font-normal">change the name of an existing position</span>
- Delete position — <span className="font-normal">remove a position from the system</span>

---

### Adding a position

To add a position, go to the <b>Positions</b> page, click <b>Create New</b> at the top of the page, and enter the position name in the modal window.
