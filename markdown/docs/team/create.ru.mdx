## Создание пользователя

Чтобы создать пользователя, перейдите на страницу <b>«Команда»</b>, нажмите кнопку <b>«Создать нового»</b> в верхней части страницы и в появившемся меню выберите пункт <b>«Пользователь»</b>.

После этого откроется модальное окно создания пользователя, в котором необходимо заполнить <b>«Полное имя»</b> и <b>«Email»</b>
