## Команда

Команда — это список пользователей внутри рабочего пространства (Workspace), которые участвуют в работе над задачами и проектами.

Пользователи не объединяются в отдельные группы — все взаимодействие происходит через сущности системы:

- Задачи — <span className="font-normal">пользователь может быть исполнителем или создателем.</span>
- Проекты — <span className="font-normal">пользователь может быть создателем проекта.</span>
- Комментарии — <span className="font-normal">пользователь выступает отправителем.</span>

Одна задача может иметь только одного исполнителя.

Каждый пользователь содержит следующие данные:

- Полное имя — <span className="font-normal">отображаемое имя пользователя.</span>
- Email — <span className="font-normal">используется для входа и связи.</span>
- Роль — <span className="font-normal">определяет уровень доступа (владелец, пользователь, гость).</span>
- Должность — <span className="font-normal">должность пользователя в команде.</span>
- Контактные данные — <span className="font-normal">телефон, адрес и другие поля.</span>
- Подтвержден ли email — <span className="font-normal">подтвердил ли пользователь email через ссылку в письме.</span>

---

### Просмотр и навигация

Пользователи отображаются в виде списка или сетки на странице <b>«Команда»</b>. Чтобы перейти в этот раздел, необходимо выбрать пункт <b>«Команда»</b> в боковой навигации приложения.

Также просмотр и редактирование пользователя доступны на странице профиля. Для этого необходимо нажать на имя пользователя и открыть его карточку. На этой странице можно изменять данные пользователя.

---

### Действия

Для пользователей доступны следующие основные действия:

- Создание пользователя — <span className="font-normal">добавление нового участника в команду.</span>
- Редактирование пользователя — <span className="font-normal">изменение данных пользователя.</span>
- Удаление пользователя — <span className="font-normal">удаление пользователя.</span>
- Изменение пароля — <span className="font-normal">обновление пароля для входа в систему.</span>
- Изменение изображения пользователя — <span className="font-normal">загрузка или обновление аватара пользователя.</span>

В зависимости от роли (владелец, пользователь, гость) эти действия ограничены и будут описаны в следующем разделе (Роли и права доступа).
