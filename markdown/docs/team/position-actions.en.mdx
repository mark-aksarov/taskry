## Position actions

Working with positions includes the following actions:

- Create position — <span className="font-normal">add a new position</span>
- Edit position — <span className="font-normal">change the name of an existing position</span>
- Delete position — <span className="font-normal">remove a position from the system</span>

---

### Adding a position

To add a position, go to the <b>“Positions”</b> page, click the <b>“Create new”</b> button at the top of the page, and in the modal window enter the position name.

#### Editing a position

To edit a position, in the <b>“Actions menu”</b> select <b>“Edit”</b>. After that, an edit window will open where you can change the position name. The position name cannot be empty.

To delete a position, in the <b>“Actions menu”</b> select <b>“Delete”</b>. In the modal window that appears, click the <b>“Delete”</b> button.
