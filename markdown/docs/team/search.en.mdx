---

## User filtering

To filter users, go to the <b>Team</b> page and click the <b>Filters</b> button at the top of the page. A filter modal will open with the following options:

- No active tasks — <span className="font-normal">shows only users without active tasks.</span>
- Has active tasks — <span className="font-normal">shows only users who have active tasks.</span>
- Has overdue tasks — <span className="font-normal">shows only users with overdue tasks.</span>
- Position — <span className="font-normal">filters users by their position.</span>

## User sorting

To sort users, go to the <b>Team</b> page and click the <b>Sort</b> button at the top of the page. The following sorting options are available:

- By full name — <span className="font-normal">sorts users by full name.</span>
- By position — <span className="font-normal">sorts users by position.</span>
