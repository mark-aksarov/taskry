# Просмотр команды

Пользователи отображаются на странице <b>Команда</b> в виде списка или карточек. На мобильных устройствах доступен только режим карточек.

Пользователь в списке и карточке может содержать следующие данные (в зависимости от типа отображения):

- Полное имя — <span className="font-normal">отображаемое имя пользователя.</span>
- Аватар — <span className="font-normal">изображение пользователя.</span>
- Email — <span className="font-normal">используется для связи и перехода к отправке письма.</span>
- Номер телефона — <span className="font-normal">контактный номер пользователя.</span>
- Публичная ссылка — <span className="font-normal">ссылка на публичный профиль пользователя.</span>
- Должность — <span className="font-normal">роль или позиция пользователя в команде.</span>

На десктопных устройствах при нажатии на <b>имя пользователя</b> открывается боковая панель с соответствующей информацией о пользователе. 

На мобильных устройствах поведение упрощено: нажатие на карточку сразу выполняет переход на страницу профиля пользователя.

Отдельная кнопка <b>меню действий</b> предоставляет доступ к дополнительным операциям.
