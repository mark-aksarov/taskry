## Documentation

Welcome to the <b>Taskry</b> documentation. Here you will find guides for working with projects, tasks, customers, and the team.

<b>Taskry</b> provides features for managing projects, tasks, users, customers, and other entities. Below is a short overview of the main sections of the application.

---

### Projects

The <b>Projects</b> section covers the main features for working with projects and project categories.

- View projects — <span className="font-normal">view the list of projects and their details</span>
- Project actions — <span className="font-normal">create, edit, change status, and delete projects</span>
- Search and filters — <span className="font-normal">search, filter, and sort projects by different parameters</span>
- Project categories — <span className="font-normal">view, create, edit, and delete project categories</span>

---

### Tasks

The <b>Tasks</b> section covers the main features for working with tasks, subtasks, and task categories.

- View tasks — <span className="font-normal">view the list of tasks and their details</span>
- Task actions — <span className="font-normal">create, edit, change status, and delete tasks</span>
- Search and filters — <span className="font-normal">search, filter, and sort tasks by different parameters</span>
- Subtasks — <span className="font-normal">create, edit, and change the status of subtasks</span>
- Task categories — <span className="font-normal">view, create, edit, and delete task categories</span>

---

### Team

The <b>Team</b> section covers the main features for managing users and positions.

- View team — <span className="font-normal">view the list of users and open their profiles</span>
- User actions — <span className="font-normal">create, edit, and delete users, as well as manage passwords and avatars</span>
- Search and filters — <span className="font-normal">search, filter, and sort users by different parameters</span>
- Positions — <span className="font-normal">view, create, edit, and delete positions</span>

---

### Customers

The <b>Customers</b> section covers the main features for working with customers and companies.

- View customers — <span className="font-normal">view the list of customers and open their profiles</span>
- Customer actions — <span className="font-normal">create, edit, and delete customers</span>
- Search and filters — <span className="font-normal">search, filter, and sort customers by different parameters</span>
- Companies — <span className="font-normal">view, create, edit, and delete companies</span>
