# Документация

Добро пожаловать в документацию <b>Taskry</b>. Здесь вы найдете руководства по работе с проектами, задачами, клиентами и командой.

<b>Taskry</b> предоставляет функционал для управления проектами, задачами,
пользователя и клиентами, а также дополнительными сущностями системы. Ниже
приведен краткий обзор основных разделов приложения.

---

## Проекты

В разделе <b>Проекты</b> описаны основные возможности по работе с проектами, а также категориями проектов.

- Просмотр проектов — <span className="font-normal">просмотр списка проектов и их детализация</span>
- Действия с проектами — <span className="font-normal">создание, редактирование, изменение статуса и удаление проектов</span>
- Поиск и фильтрация — <span className="font-normal">поиск, фильтрация и сортировка проектов по различным параметрам</span>
- Категории проектов — <span className="font-normal">просмотр, создание, редактирование и удаление категорий проектов</span>

---
