## Project categories

A project category is a way to group projects by type or topic. Each category has a unique name. A project can only be assigned to one category.

---

### Viewing categories

Project categories are shown in dropdown lists when you create or edit a project. They are also available on the categories page and used for filtering.

To view all categories, go to the <b>“Projects”</b> page, click the <b>“Manage”</b> button at the top of the page, and select <b>“Categories”</b> from the menu to open the <b>“Project categories”</b> page.
