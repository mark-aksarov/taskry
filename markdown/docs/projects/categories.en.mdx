# Project categories

A project category is a way to group projects by type or topic. Each category has a unique name. A project can belong to only one category.

---

## Viewing categories

Project categories are displayed in dropdown lists when creating or editing a project. They are also used for filtering.

To view all categories, go to the <b>Projects</b> page, click <b>Manage</b> at the top of the page, and select <b>Categories</b> to open the <b>Project categories</b> page.

## Category actions

Working with project categories includes the following actions:

- Create a project category — <span className="font-normal">add a new project category</span>
- Edit a project category — <span className="font-normal">change the name of an existing category</span>
- Delete a project category — <span className="font-normal">remove a category from the system</span>

---

### Creating a category

To create a category on the <b>Project categories</b> page, click <b>Create new</b> at the top of the page. In the modal modal, enter the category name and confirm creation.

### Deleting a category

To delete a category, open the <b>Actions menu</b> and select <b>Delete</b>. All related projects will also be deleted. Deleting a category cannot be undone, so make sure the category is no longer needed and all important data has been saved before deleting it.
