# Проекты

Проект — это сущность, которая хранит основную информацию о работе и объединяет связанные с ней данные:

- Основная информация — <span className="font-normal">название, описание, дедлайн</span>
- Задачи — <span className="font-normal">единицы работы внутри проекта</span>
- Комментарии — <span className="font-normal">обсуждения, привязанные к проекту</span>
- Клиент — <span className="font-normal">информация о клиенте, для которого выполняется проект</span>
- Категория — <span className="font-normal">группировка проекта по типу или направлению</span>
- Статус — <span className="font-normal">текущее состояние проекта</span>
- Кто создал — <span className="font-normal">пользователь, создавший проект</span>

---

## Краткий обзор

В этом разделе рассматривается работа с проектами и основные возможности, которые доступны пользователю в системе.

### Просмотр проектов

Здесь описывается, как проекты отображаются в интерфейсе системы. Рассматриваются два формата представления — список и карточки, а также то, какую краткую информацию пользователь может увидеть о проекте.

### Действия с проектами

Здесь рассматриваются основные действия с проектами: создание новых проектов, редактирование проектов, изменение статуса и удаление проектов.

### Поиск и фильтрация

Здесь объясняется, как пользователь может находить нужные проекты среди большого количества записей, а также фильтрация и сортировка, которые помогают сузить выборку.

### Категории проектов

Здесь рассматриваются категории проектов, а также какие действия доступны пользователю — просмотр списка категорий, их создание, редактирование и удаление. Категории помогают структурировать проекты и упрощают поиск нужных проектов внутри системы.
