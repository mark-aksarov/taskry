# Просмотр проектов

Проекты отображаются на странице <b>Проекты</b> в виде списка или карточек. На мобильных устройствах доступен только режим карточек.

Проект в списке и карточке может содержать следующие данные (в зависимости от типа отображения):

- Заголовок — <span className="font-normal">краткое название проекта</span>
- Срок выполнения — <span className="font-normal">срок завершения проекта</span>
- Статус — <span className="font-normal">текущее состояние проекта</span>
- Кем создан — <span className="font-normal">изображение или полное имя пользователя, создавшего проект</span>
- Клиент — <span className="font-normal">изображение или полное имя клиента</span>
- Категория — <span className="font-normal">группа для организации проектов</span>
- Компания — <span className="font-normal">название организации клиента</span>
- Прогресс — <span className="font-normal">отображение выполнения задач проекта в визуальной шкале</span>

На десктопных устройствах при нажатии на <b>заголовок проекта</b>, <b>имя автора</b> или <b>имя клиента</b> открывается боковая панель с соответствующей информацией (о проекте, пользователе или клиенте). 

На мобильных устройствах поведение упрощено: нажатие на карточку сразу выполняет переход на страницу детализации проекта.

Отдельная кнопка <b>комментарии</b> открывает окно с комментариями проекта, а <b>меню действий</b> предоставляет доступ к дополнительным операциям.
