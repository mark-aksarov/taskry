## Редактирование проекта

Чтобы отредактировать проект, перейдите на страницу <b>«Проекты»</b>, нажмите кнопку <b>«Меню действий»</b> (иконка с тремя точками) справа в карточке или в элементе списка и в появившемся меню выберите пункт <b>«Редактировать»</b>.

После этого откроется окно редактирования, в котором можно изменить данные проекта. Обязательные поля (заголовок и дедлайн) не могут быть пустыми.

Редактирование проекта включает изменение следующих данных:

- Заголовок — <span className="font-normal">краткое название проекта</span>
- Дедлайн — <span className="font-normal">срок завершения проекта</span>
- Статус — <span className="font-normal">текущее состояние проекта</span>
- Описание — <span className="font-normal">дополнительная информация о проекте</span>
- Клиент — <span className="font-normal">кто заказчик или для кого выполняется проект</span>
- Категория — <span className="font-normal">группа для удобной организации проектов</span>

После сохранения изменения сразу отобразятся в списке проектов.
