# Категории проектов

Категория проекта — это способ группировки проектов по типу или тематике. Каждая категория имеет уникальное название. Проект может быть назначен только в одну категорию.

---

## Просмотр категорий

Категории проектов отображаются в выпадающих списках при создании или редактировании проекта. Также они используются при фильтрации.

Чтобы просмотреть все категории, перейдите на страницу <b>Проекты</b>, нажмите кнопку <b>Управление</b> в верхней части страницы и выберите пункт меню <b>Категории</b>, чтобы перейти на страницу <b>Категории проектов</b>.

## Действия с категориями

Работа с категориями проектов включает следующие действия:

- Создание категории проекта — <span className="font-normal">добавление новой категории проектов</span>
- Редактирование категории проекта — <span className="font-normal">изменение названия существующей категории</span>
- Удаление категории проекта — <span className="font-normal">удаление категории из системы</span>

---

### Добавление категории

Чтобы добавить категорию на странице <b>Категории проектов</b>, нажмите кнопку <b>Создать новую</b> в верхней части страницы. В появившемся модальном окне введите название категории проекта и подтвердите создание категории.

### Удаление категории

Чтобы удалить категорию в <b>Меню действий</b> выберите пункт <b>Удалить</b>. Вместе с категорией будут удалены все связанные проекты. Удаление категории является необратимым действием, поэтому перед удалением рекомендуется убедиться, что категория больше не нужна и все важные данные сохранены.
