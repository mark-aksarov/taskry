# Project actions

This section describes the main project actions: creating, editing, changing status, and deleting projects.

---

## Creating a project

To create a project on the <b>Projects</b> page, click <b>Create new</b>, select <b>Project</b>, fill in the required fields, and confirm project creation.

Creating a project includes entering the following information:

- Title — <span className="font-normal">short project name</span>
- Deadline — <span className="font-normal">project completion date</span>
- Status — <span className="font-normal">current project state</span>
- Description — <span className="font-normal">additional information about the project</span>
- Customer — <span className="font-normal">who requested the project or who it is for</span>
- Category — <span className="font-normal">group used to organize projects</span>

The <b>Title</b> and <b>Deadline</b> fields are required.

After saving, the project will be created successfully.

## Deleting a project

The system supports two types of deletion: <b>single</b> and <b>bulk</b>. On mobile devices, only <b>single</b> deletion is available. Both options permanently remove projects from the system.

Deleting a project cannot be undone, so make sure the project is no longer needed and all important data has been saved before deleting it.

### Single deletion

To delete a project on the <b>Projects</b> page, click the <b>Actions menu</b> button (three-dot icon) in the project card or list item and select <b>Delete</b>.

## Changing project status

The system supports two types of status changes: <b>single</b> and <b>bulk</b>. On mobile devices, only <b>single</b> status changes are available.

### Single status change

To change a project status on the <b>Projects</b> page, click the <b>Actions menu</b> button (three-dot icon) in the project card or list item and select a new project status.
