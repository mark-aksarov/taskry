# Search and filtering

This section explains how users can find the projects they need among many records, as well as <b>filtering</b> and <b>sorting</b>, which help narrow down the results.

---

## Filtering projects

To filter projects, go to the <b>Projects</b> page and click <b>Filters</b> at the top of the page. A filter modal will open with the following options:

- No active tasks — <span className="font-normal">shows only projects without active tasks.</span>
- Deadline from / to — <span className="font-normal">allows you to select a date range.</span>
- Status — <span className="font-normal">filter by project state: <b>Pending</b>, <b>Active</b>, <b>Completed</b>.</span>
- Category — <span className="font-normal">filter and group projects by category.</span>
- Customer — <span className="font-normal">filter by customers linked to projects.</span>
- Created by — <span className="font-normal">filter by users who created projects.</span>

## Sorting projects

To sort projects, go to the <b>Projects</b> page and click <b>Sorting</b> at the top of the page. The following sorting options are available:

- By creation date — <span className="font-normal">sort from newest to oldest.</span>
- By title — <span className="font-normal">sort in alphabetical order.</span>
- By deadline — <span className="font-normal">sort by the nearest deadline.</span>
- By status — <span className="font-normal">sort by project state: <b>Active</b>, <b>Pending</b>, <b>Completed</b>.</span>
