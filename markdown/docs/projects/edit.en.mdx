## Editing a project

To edit a project, go to the <b>“Projects”</b> page, click the <b>“Actions menu”</b> button (three dots icon) on the right side of the card or list item, then select <b>“Edit”</b> from the menu that appears.

Project editing includes updating the following fields:

- Title — <span className="font-normal">short name of the project</span>
- Deadline — <span className="font-normal">project completion due date</span>
- Status — <span className="font-normal">current state of the project</span>
- Description — <span className="font-normal">additional information about the project</span>
- Client — <span className="font-normal">who the customer is or for whom the project is being done</span>
- Category — <span className="font-normal">a group for organizing projects more conveniently</span>

After saving, the changes will be immediately reflected in the project list.
