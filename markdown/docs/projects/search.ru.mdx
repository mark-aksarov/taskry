# Поиск и фильтрация

Здесь объясняется, как пользователь может находить нужные проекты среди большого количества записей, а также <b>фильтрация</b> и <b>сортировка</b>, которые помогают сузить выборку.

---

## Фильтрация проектов

Чтобы отфильтровать проекты, перейдите на страницу <b>Проекты</b>, нажмите кнопку <b>Фильтры</b> в верхней части страницы. Откроется модальное окно фильтрации, где доступны следующие фильтры:

- Нет активных задач — <span className="font-normal">отображает только проекты без активных задач.</span>
- Дедлайн с / по — <span className="font-normal">позволяет выбрать диапазон дат.</span>
- Статус — <span className="font-normal">фильтрация по состоянию проекта: <b>В ожидании</b>, <b>Активный</b>, <b>Завершён</b>.</span>
- Категория — <span className="font-normal">позволяет фильтровать и группировать проекты по категориям.</span>
- Клиент — <span className="font-normal">фильтрация по клиентам, связанных с проектами.</span>
- Кем создан — <span className="font-normal">фильтрация по пользователям, создавших проекты</span>

## Сортировка проектов

Чтобы отсортировать проекты, перейдите на страницу <b>Проекты</b>, нажмите кнопку <b>Сортировка</b> в верхней части страницы. В открывшемся меню доступны следующие варианты сортировки:

- По дате создания — <span className="font-normal">сортировка от новых к старым.</span>
- По заголовку — <span className="font-normal">сортировка в алфавитном порядке.</span>
- По сроку — <span className="font-normal">сортировка по ближайшему дедлайну.</span>
- По статусу — <span className="font-normal">сортировка по состоянию проекта: <b>Активный</b>, <b>В ожидании</b>, <b>Завершён</b>.</span>
