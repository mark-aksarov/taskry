# Действия с проектами

В этом разделе описаны основные действия с проектами: создание, редактирование, изменение статуса и удаление.

---

## Создание проекта

Для создания проекта на странице <b>Проекты</b> нужно нажать кнопку <b>Создать новый</b>, выбрать пункт <b>Проект</b>, заполнить необходимые поля в форме и подтвердить создание проекта.

Создание проекта включает заполнение следующих данных:

- Заголовок — <span className="font-normal">краткое название проекта</span>
- Срок выполнения — <span className="font-normal">срок завершения проекта</span>
- Статус — <span className="font-normal">текущее состояние проекта</span>
- Описание — <span className="font-normal">дополнительная информация о проекте</span>
- Клиент — <span className="font-normal">кто заказчик или для кого выполняется проект</span>
- Категория — <span className="font-normal">группа для удобной организации проектов</span>

Поля <b>Заголовок</b> и <b>Срок выполнения</b> являются обязательными для заполнения. 

После сохранения проект будет успешно создан.

## Удаление проекта

В системе предусмотрено два типа удаления: <b>одиночное</b> и <b>массовое</b>. На мобильных устройствах доступно только <b>одиночное</b> удаление. Оба варианта позволяют безвозвратно удалить проекты из системы.
Удаление проекта является необратимым действием, поэтому перед удалением рекомендуется убедиться, что проект больше не нужен и все важные данные сохранены.

### Одиночное удаление

Чтобы удалить проект на странице <b>Проекты</b> нажмите кнопку <b>Меню действий</b> (иконка с тремя точками) в карточке или списке проекта и выберите пункт <b>Удалить</b>.
