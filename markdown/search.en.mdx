## Search

The search function in Taskry allows you to quickly find the tasks, projects, and comments you need.

---

### How to Use Search?

Search is a powerful tool for navigating large amounts of data.

---

### Main Search Features

Find information efficiently:

#### 1. Quick Search

Use the search bar at the top of the interface to search across all projects.

#### 2. Search Filters

Refine results using filters: status, priority, assignee, date.

#### 3. Saved Searches

Save frequently used search queries for quick access.

---

### Search Tips

Use keywords and operators for more accurate search. You can search by task title, description, and comments.
