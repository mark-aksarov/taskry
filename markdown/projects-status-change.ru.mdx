## Изменение статуса

В системе предусмотрено два типа изменения статуса: <b>одиночное</b> и <b>массовое</b>.

---

### Одиночное изменение статуса

Чтобы изменить статус проекта, нажмите кнопку <b>«Меню действий»</b> (иконка с тремя точками) справа у карточки или элемента списка проекта и в появившемся меню выберите новый статус проекта.

![image](https://storage.yandexcloud.net/task-manager-docs/project_single_status_change_ru.jpg)

После выбора статус обновится для выбранного проекта.

---

### Массовое изменение статуса

Чтобы изменить статус нескольких проектов одновременно, отметьте нужные проекты с помощью чекбоксов. Затем нажмите кнопку <b>«Действия»</b> и выберите новый статус проектов.

![image](https://storage.yandexcloud.net/task-manager-docs/project_multiple_status_change_ru.jpg)

После выбора статус обновится для выбранных проекта.
