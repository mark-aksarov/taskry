## Отображение проектов

Система поддерживает два режима отображения проектов. В режиме <b>«Список»</b> данные выводятся с упорядоченными строками и колонками. В режиме <b>«Карточки»</b> проекты представлены в виде сетки карточек, где каждый элемент отображается как отдельный блок. Оба режима используют один и тот же набор данных, но различаются способом визуализации.

---

### Список

Чтобы переключиться в режим списка, нажмите кнопку <b>«Список»</b> в панели переключения вида в верхней части страницы.

![image](https://storage.yandexcloud.net/task-manager-docs/project_list_ru.jpg)

Каждый элемент в списке содержит следующие данные

- Заголовок проекта — <span className="font-normal">краткий заголовок</span>
- Дедлайн — <span className="font-normal">дата завершения</span>
- Кем создан — <span className="font-normal">кто создал проект</span>
- Клиент — <span className="font-normal">связанный заказчик</span>
- Категория — <span className="font-normal">тип проекта</span>
- Компания — <span className="font-normal">организация клиента</span>
- Статус — <span className="font-normal">текущее состояние</span>

---

### Карточки

Чтобы переключиться в режим списка, нажмите кнопку <b>«Карточки»</b> в панели переключения вида в верхней части страницы.

![image](https://storage.yandexcloud.net/task-manager-docs/project_grid_ru.jpg)

Каждая карточка проекта содержит следующие данные

- Заголовок проекта — <span className="font-normal">краткий заголовок</span>
- Дедлайн — <span className="font-normal">дата завершения</span>
- Статус — <span className="font-normal">текущее состояние</span>

---

### Интерактивные действия

- Нажатие на заголовок проекта
  <div>
    Открывает боковую панель с деталями проекта: описание, дедлайн, статус,
    категория, клиент и создатель. Позволяет быстро просмотреть полную
    информацию без перехода на отдельную страницу.
  </div>
- Нажатие на имя создателя проекта
  <div>
    Открывает боковую панель пользователя: должность, контакты, краткое описание
    и другие сведения. Помогает быстро понять роль сотрудника и при
    необходимости перейти к его профилю.
  </div>
- Нажатие на имя клиента
  <div>
    Открывает боковую панель клиента: контактные данные, компания, краткое
    описание и ключевая информация. Позволяет получить контекст о клиенте и
    перейти к его карточке.
  </div>
- Кнопку «Комментарии»
  <div>
    При нажатии на <b>«Комментарии»</b> открывается модальное окно с
    комментариями проекта.
  </div>
- Меню действий
  <div>
    При нажатии на кнопку <b>«Меню действий»</b> открывается список доступных
    действий
  </div>
