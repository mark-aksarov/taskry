## Сортировка проектов

Чтобы отсортировать проекты, нажмите кнопку <b>«Сортировка»</b> в верхней панели списка. В открывшемся меню доступны следующие варианты сортировки:

- По дате создания — <span className="font-normal">от новых к старым или наоборот.</span>
- По названию — <span className="font-normal">в алфавитном порядке.</span>
- По сроку — <span className="font-normal">по ближайшему или самому позднему дедлайну.</span>
- По статусу — <span className="font-normal">группировка по состоянию проекта: «Активный», «В ожидании», «Завершён».</span>

![image](https://storage.yandexcloud.net/task-manager-docs/project_sorting_ru.jpg)

После выбора параметра список проектов обновится и будет отображён в соответствии с выбранным способом сортировки.
