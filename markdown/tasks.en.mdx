## Tasks

Tasks are the basic work units in Taskry that help you track work progress.

---

### What is a Task?

A task represents a single piece of work that needs to be completed. Each task can have a priority, deadline, assignee, and many other attributes.

---

### Creating a Task

Create tasks to track your work:

#### 1. Open a Project

Select the required project from the list.

#### 2. Click "Add Task"

Enter the task title and description.

#### 3. Set Parameters

Assign an executor, set priority and deadline.

---

### Task Statuses

Tasks can have different statuses: new, in progress, under review, completed. Use statuses to track progress.
