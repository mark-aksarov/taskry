## Поиск

Функция поиска в Taskry позволяет быстро найти нужные задачи, проекты и комментарии.

---

### Как использовать поиск?

Поиск — это мощный инструмент для навигации по большому количеству данных.

---

### Основные возможности поиска

Эффективно ищите информацию:

#### 1. Быстрый поиск

Используйте поисковую строку в верхней части интерфейса для поиска по всем проектам.

#### 2. Фильтры поиска

Уточните результаты с помощью фильтров: статус, приоритет, исполнитель, дата.

#### 3. Сохранённые поиски

Сохраняйте часто используемые поисковые запросы для быстрого доступа.

---

### Советы по поиску

Используйте ключевые слова и операторы для более точного поиска. Вы можете искать по названию задачи, описанию и комментариям.
