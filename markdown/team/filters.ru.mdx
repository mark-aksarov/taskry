## Фильтры пользователей

Чтобы отфильтровать пользователей, перейдите на страницу <b>«Команда»</b>, нажмите кнопку <b>«Фильтры»</b> в верхней части страницы. Откроется модальное окно фильтрации, где доступны следующие фильтры:

- Нет активных задач — <span className="font-normal">отображает только пользователей без активных задач.</span>
- Есть активные задачи — <span className="font-normal">отображает только пользователей, у которых есть активные задачи.</span>
- Есть просроченные задачи — <span className="font-normal">отображает только пользователей с просроченными задачами.</span>
- Должность — <span className="font-normal">фильтрация пользователей по занимаемой должности.</span>
