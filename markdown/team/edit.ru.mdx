## Редактирование пользователя

Чтобы отредактировать пользователя, перейдите на страницу <b>«Команда»</b>, нажмите кнопку <b>«Меню действий»</b> (иконка с тремя точками) справа в карточке или в элементе списка и в появившемся меню выберите пункт <b>«Редактировать»</b>.

После этого откроется модальное окно редактирования, в котором можно изменить данные пользователя. <b>«Полное имя»</b> не может быть пустыми.

Редактирование пользователя включает изменение следующих данных:

- Полное имя — <span className="font-normal">отображаемое имя пользователя.</span>
- О себе - <span className="font-normal">краткая информация о пользователе, отображаемая в профиле.</span>
- Дата рождения - <span className="font-normal">используется для хранения даты рождения пользователя.</span>
- Номер телефона - <span className="font-normal">контактный номер пользователя для связи.</span>
- Публичная ссылка - <span className="font-normal">уникальная ссылка на профиль пользователя, доступная другим пользователям.</span>
- Адрес - <span className="font-normal">местоположение или адрес пользователя.</span>
- Должность — <span className="font-normal">должность пользователя в команде.</span>

После сохранения изменения сразу отобразятся в списке пользователей.
