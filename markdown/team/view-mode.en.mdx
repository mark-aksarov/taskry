## Team view

The system supports two team view modes. In <b>“List”</b> mode, data is displayed as a list of items. In <b>“Cards”</b> mode, tasks are presented in a grid, where each user is shown as a separate card. Both modes use the same dataset but differ in visualization.

---

### List

To switch to list mode, go to the <b>“Team”</b> page and click the <b>“List”</b> button in the view switcher at the top of the page.

- Email — <span className="font-normal">used for communication and opening an email client.</span>

### Actions

- Clicking on the user’s name or avatar <span className="font-normal">opens the user sidebar: position, contacts, short description, and other details. Helps quickly understand the user’s role and navigate to their profile if needed.</span>
- Clicking on the email <span className="font-normal">opens the email client via a mailto link.</span>
- Clicking on the phone number <span className="font-normal">initiates a call via a tel link on supported devices.</span>
- Clicking on the public link <span className="font-normal">opens the user’s profile page in a new tab.</span>
- Clicking on the “Actions menu” <span className="font-normal">opens the list of available actions.</span>
