## Просмотр команды

Система поддерживает два режима отображения команды. В режиме <b>«Список»</b> данные выводятся в виде элементов списка. В режиме <b>«Карточки»</b> задачи представлены в виде сетки, где каждый пользователь отображается как отдельная карточка. Оба режима используют один и тот же набор данных, но различаются способом визуализации.

---

### Список

Чтобы переключиться в режим списка, перейдите на страницу <b>«Команда»</b>, нажмите кнопку <b>«Список»</b> в панели переключения вида в верхней части страницы.

- Аватар — <span className="font-normal">изображение пользователя.</span>

- Публичная ссылка — <span className="font-normal">ссылка на публичный профиль пользователя.</span>
- Должность — <span className="font-normal">роль или позиция пользователя в команде.</span>

### Действия

- Нажатие на имя пользователя или аватар <span className="font-normal">открывает боковую панель пользователя: должность, контакты, краткое описание и другие сведения. Помогает быстро понять роль сотрудника и при необходимости перейти к его профилю.</span>
- Нажатие на email <span className="font-normal">открывает почтовый клиент через mailto-ссылку для отправки письма.</span>
- Нажатие на номер телефона <span className="font-normal">инициирует звонок через tel-ссылку на поддерживаемых устройствах.</span>
- Нажатие на публичную ссылку <span className="font-normal">открывает страницу профиля пользователя в новой вкладке.</span>
- Нажатие на «Меню действий» <span className="font-normal">открывает список доступных действий.</span>
