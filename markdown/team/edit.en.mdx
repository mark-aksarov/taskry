## Editing a user

To edit a user, go to the <b>“Team”</b> page, click the <b>“Action menu”</b> button (three dots icon) on the right side of a card or list item, and select <b>“Edit”</b> from the dropdown menu.

Editing a user includes updating the following data:

- Full name — <span className="font-normal">the user's display name.</span>
- About — <span className="font-normal">short information about the user shown in the profile.</span>
- Date of birth — <span className="font-normal">used to store the user's birth date.</span>
- Phone number — <span className="font-normal">a contact number for communication.</span>
- Public link — <span className="font-normal">a unique link to the user's profile available to others.</span>
- Address — <span className="font-normal">the user's location or address.</span>
- Position — <span className="font-normal">the user's role or job title in the team.</span>

After saving, the changes will be immediately reflected in the user list.
